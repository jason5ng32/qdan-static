  1. 非越狱

  2. VPN

    虚拟专用网，iOS 默认就支持 L2TP、PPTP、IPsec，可以搭配 App 日常使用，如 Cisco anyconnect（这个其实我也说不清，这个官方说其实是个客户端，但是有的 VPN 提供商额外给这个选项）。

    据说现在对 VPN 干扰越来越强了，当然我这个算是谣言，因为我没考证过。

    虛擬私人網路 - 维基百科，自由的百科全书
      zh.wikipedia.org

    • HTTP 代理

      通过局域网内其他设备来共享，修改 Wi-Fi 设置下 HTTP 代理实现，之所以说软件是因为 Shadowsocks 也支持这种方法，只不过后台有可能会被干掉，需要偶尔切回来维持后台。

    • 描述文件

      通过一个描述文件解决蜂窝数据下没有 HTTP 代理的问题，也有叫“APN”、"APNP"的，具体我也说不太清楚了（汗），速度和体验与费用成正比，比如某音乐加速服务能看高清油土鳖，大多与 HTTP 代理方式捆绑提供，算是 Wi-Fi 和 3G 下的共同解决方案吧，毕竟花钱买的。

    • DNS

      应该是最简单、省电的方式了，之前有免费提供过，然后被迅速扑灭，现在也有一些收费的在做，价格不等。

    • 香港 3G 卡际漫游

      任性的选择，一直没亲自试过，就不多说了……

    • 越狱

      支持上述非越狱所有方式外的额外方法。

    • Shadowsocks

      2013年出现的新方法，基于 socks5，跨平台，越狱下无需考虑后台问题，整体感觉非常好，最重要的是，现在活的好好的，若说缺点的话，需要稳定服务器，要有一定经济投入。

      Shadowsocks - 维基百科，自由的百科全书
        zh.wikipedia.org

      • GoAgent

        据某网站统计“2014年10大最受网友欢迎翻墙软件”，综合排名第二，感觉现在也开始不给力了。

        GoAgent - 维基百科，自由的百科全书
          zh.wikipedia.org

        • Hosts

          应该是除 DNS 外最省电的方案了吧，如果人品够好，你找到的 IP 一直可用，那么一次设定就不需要再改了，但是现在基本废掉了吧。

          iOS：/private/etc
