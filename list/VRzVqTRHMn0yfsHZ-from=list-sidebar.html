美剧轻单精选

豆瓣美剧前二十页的个人感受

知乎上面“哪些美剧过誉了？”问题中的回答，全篇都没什么营养。全是主观感受的简略概括，很多剧其实该花很多篇幅讲讲的，but我懒。刷剧无涯，做字幕是岸啊。

  1. 回答本身

    根据各大奖项来看，真的烂剧在美国就算收视率高也没得过什么奖，过誉的情况应该不是很多吧。虽然有很多得奖的剧我个人感觉无聊致死，但是，其实我啥也不懂，so......

     

    TBBT真的很好玩儿的时候好像没像现在这样成中国国民喜剧，现在就…...而且艾美连续得喜剧最佳男主实在是有点勉强了吧。以目前的水准看，《生活大爆炸》是一部严重过誉的美剧。《破产姐妹》联想力很强，什么都能拿来当黄段子或者歧个视吐个槽，拿来下饭的时候看也不错。《吸血鬼日记》之类的…这种剧火是应该的，观众群摆在那里，狗血也是应该的，编这么长不容易。

     

    《老友记》没有被过誉，显然嘛，具体多厉害，很多答案说得比我好多了。

     

    《老爸老妈浪漫史》是我挚爱啊，不包括最后两集。

    哪些美剧过誉了？ - 舒海萌的回答 - 知乎
      zhihu.com

    • Page1-3

      page 1：前十里面八个是《friends》，经典属性加中国观众尿性，不多说了，同页还有成长的烦恼，一个意思；还有南方公园和辛普森一家，动画片，这儿不讨论啦（同样电视节目或者真人秀也不讨论）；同页还有权利的游戏第五季，还没开播呢拜托。

       

      page 2：《Breaking bad》，什么？在知乎你敢说白粉一个不字？！《shameless》season 5，Shameless打多少分都行啦啦啦；《Game of Thrones》season 4，nice；《Will and Grace》nice；《the good wife》so very fucking nice；《law and order》。。。时至今日，我脑子里对《法律与秩序》的印象就是“咚咚”；《person of interest》。。。fine；

       

      page 3：《风骚律师》，again，在在知乎你敢说白粉一个不字？！《modern family》是很厉害，不至于空前绝后，就凭那么多奖，多年以后也会被归入经典咯，没有特别喜欢看的喜剧就看看这个吧；《Louis》一集弃，不是说不好，应该有点屌炸天，但是没办法我兴趣不高，我怀疑很多人兴趣不高；

    • Page4-8

      page 4：《mad man》没什么好说的咯，好看炸嘛，只要忍得了男主角，海阔天空；《newsroom》对于字幕组来说算神剧吧，翻译起来应该很那啥呢，对于大部分观众也就是“据说超厉害哎”然后迷迷糊糊被教育的话唠电视剧吧，我觉得这压根就是喜剧啊台词跟人物都挺逗的；

       

      page 5：《豪斯医生》一集弃，我知道你们喜欢，别回了谢谢我可能过段时间也喜欢；《TBBT》season 1，唉，是很好看；《大西洋帝国》很棒，片头曲好听，我看了几集忘了继续看了就始终没看完，回头补吧；

       

      page 6：都是之前提过的

       

      page 7：《Boston legal》男性角色很招人喜欢，女性角色就有点儿......

       

      page 8：《发展受阻》《办公室》《宋飞》绝对经典，虽然我全一集弃了，我猜单纯是我品味差，不是剧不好。

    • Page9-11

      page 9：《马尔科姆一家》again，一集弃，我猜还是我品味差；《白宫风云》，绝对好看，但我没找到满意画质的版本所以一集也没看过，我猜好看得死去活来你信吗？《黑道家族》，这居然才被排到第九页，这部剧的质感我很喜欢，用质感这个词，实在是因为我不知道还有什么词好形容啦。

       

      page 10：《副总统》第一集我看了三次，因为分分钟想弃，但不愿相信自己品味差，anyway最终结果是至今第一集没看完；

       

      page 11：《CSI》......就不说什么了吧，我没有看多少，又是很多年以前看的啦；《girls》不喜欢，不知道怎么评价，勉强支撑第一集；《奥丽芙基特里奇》很棒，很棒，几乎都要去看原著；《纸牌屋》season 2，呃，嗯，呃，呃，起码看上去很厉害，对了，《纸牌屋》真的没有过誉么，我总觉得时刻透着一种逼装大发了的假劲，今年的第三季愣是吊着口气没刷完；

    • Page12-14

      page 12：《community》9.2分的废柴啊，啥也不说了，一把辛酸泪；《white collar》看了第一集，不是说弃了，应该当时看完了再看别的然后就忘了继续看了于是；《寻》，也是看了一集就忘了追，但是大家要记住，人家跟欲望都市其实完全不一样的格调，不要听什么gay的欲望都市之类的噱头；

       

      page 13：《sex and the city》season 6，虽然特别恶心Carrie，不过欲望都市实在是人生无法割舍的电视剧啊，怎么都要进必选系列，多么......只要不看电影，起码别看第二部，这个系列就永远珍贵又可爱哦......

       

      page 14：以它的热门程度，《行尸走肉》才排到14页，我猜是乐意泡在豆瓣打分的大部分是女生，女生也许不怎么喜欢？不知道，反正我不喜欢哈哈。《true detective》。。。除了俩男主和炫酷镜头以外，十分不好看，HBO出品也不好看，奖多得手软也不好看，片头曲好听；《燃情克利夫兰》sorry也是没有啥动力追。

    • Page15-20

      page 15：《希区柯克剧场》，听名字就可以打高分了；《犯罪心理》，呃，没怎么看过哎；《美国恐怖故事》第五季，哈哈别做梦了，美恐真是徒有虚名，一集集拖着你看，既不恐怖也不良心，编剧一集比一集瞎逼逼，人物树不起来情节支离破碎，回头想想，这都尼玛什么鬼，当然出来还是一集集跟着看......

       

      page 16：《越狱》season 1，一开始挺好看的；《himym》season 2，唉，没有那个破结局，第九季再差，你在我心中都是满分；《戏里戏外》哈哈一集弃，我品味差。

       

      page 17：都是之前提过的

       

      page 18：《实习医生格蕾》，祝你们细水长流看下去：），我没有

       

      page 19：《bones》没有太大感觉；《法律与秩序：特殊受害者》still，咚咚！

       

      page 20：《硅谷》挺好看的。《firefly》，请感受我的敬意，太他妈，我估计也就喜欢过这一部跟科幻沾边的电视剧，太好看了，一开始听起来难听得要死的片头曲，最后都喜欢上了。
