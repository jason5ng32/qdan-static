我常用的 15 个 Sketch Plugins

Sketch 是我工作中使用频率最高的应用，有必要列个轻单来记录下 Sketch 里让我提升工作效率的几个常用插件。依据个人使用习惯，以下排名分先后。

  1. Content Generator Sketch Plugin

    快速创建头像、名字、地理位置等 mock up 的必备神器。

    timuric/Content-generator-sketch-plugin
    Content-generator-sketch-plugin - Sketch app plugin for generating dummy data such as avatars, names, photos, geo data etc
      github.com

    • Sketch Commands

      功能强大的命令工具，其中我最常用的两个功能是：以 100px 为单位快速移动图层、新建自定义大小的对象（配合自定义快捷键用起来很便捷）

      bomberstudios/sketch-commands
      sketch-commands - A collection of script commands for Bohemian Coding's Sketch.app
        github.com

      • sketch dynamic button

        跟随文本的长度，以设定好的 padding 值来变换按钮大小，这个超爽。

      • Sketch Plugin Scripts

        复制第一个选定对象，并平均分布在所选对象二者之间；以画板的宽度或高度填充所选对象，也是我常用的功能之一。

        makzan/Sketch-Plugin-Scripts
        Sketch-Plugin-Scripts - Some scripts for the Mac Sketch app
          github.com

        • Duplicator

          一次或多次复制当前图层到指定的地方。

          turbobabr/duplicator
          duplicator - A small Sketch 3 plugin that allows quickly duplicate selected layers in a specified direction.
            github.com

          • Rename It

            批量重命名图层的工具，支持图层按序号排序及倒序，还可以按当前图层宽度或高度命名。

            rodi01/RenameIt
            RenameIt - Sketch plugin to rename layers like a boss
              github.com

            • Zeplin

              团队协作工具，可把设计稿上传到 Zeplin 然后被邀请人可在项目列表中查看文件元素的各种属性但不会因此破坏到你的源文件，设计师与前端沟通的有效利器，从此不需要标注啦。

              Zeplin
              Collaboration app for UI designers and frontend developers
                zeplin.io

              • Sketch Measure

                如果忽略上一条，那么这款快速标注工具还是挺爽的，测量对象的大小、颜色、字体及对象间距一应俱全。不过我发现在最新的 3.2 版上有 bug ，有同学遇到跟我一样的问题么？

              • Sort Layers

                重新排序图层/画板，如果你跟我一样马虎，经常搞错画板的顺序，那么你需要它。

                getflourish/Sketch-Sort-Layers
                Sketch-Sort-Layers - This plugin lets you sort layers by their content.
                  github.com

                • Sketch States Switch

                  保存和切换页面几种状态的工具，可以用来给你的产品经理做演示。

                • Sketch Divine Proportions Plugin

                  Sketch 的黄金分割工具。

                  Sketch Divine Proportions Plugin by ichord
                    ichord.github.io

                  • Export to ICNS

                    把画板或切片导出为 ICNS 格式。

                    solicomo/export-to-icns
                    export-to-icns - A Sketch plugin which can export artboards or slice to ICNS.
                      github.com

                    • Sketch Dockpreview

                      可快速在你的 Dock 中预览画板中的内容，适合设计 APP 的图标时使用。

                    • Sketch Vertical Teleporter

                      垂直移动所选对象到你的光标所在位置。

                      turbobabr (Andrey Shakhmin)
                      turbobabr has 21 repositories written in JavaScript, Objective-C, and CSS. Follow their code on GitHub.
                        github.com

                      • StickyGrid

                        按网格来纠正你画跑偏的路径。
