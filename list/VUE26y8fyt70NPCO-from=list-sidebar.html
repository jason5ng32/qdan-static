在线教育网站整理

个人尝试过并坚持使用的在线教育类网站。专业关系，全部是工程和计算机类。纯个人体验，所以一些少见的或者没能坚持使用的网站就没有收录。没有排名，但分先后。

 

保持更新中。

  1. 短篇文字教程 + 即时编码练习

    根据章节或者主题分类，看一段文字，学习相应知识点，然后立刻着手编程获得反馈。

  2. Codecademy

    Learn by doing 第一站。全部是网页编程技术，偏重前端。

  3. 实验楼

    以下三家粗看界面和功能都差不多，不过因为没有深入尝试，有可能有各自不同的亮点。

    实验楼 - 第一家以实验为核心的IT在线教育平台
    实验楼是国内第一家以实验为核心的IT在线教育平台，创新的实验驱动教学模式，以就业为导向，为IT相关专业的在校学生及从业者提供编程、运维、测试、云计算、大数据、数据库等全面的IT技术动手实践环境，提供Linux、Python、Java、C语言、Ruby、Android、IOS开发等热门课程。
      shiyanlou.com

    • 萌码

    • 计蒜客

    • 短视频教程 + 即时编码练习

      以视频教程取代文字，同样也是讲解知识点后立刻动手编程。

    • Code School

      付费网站，按月或年收取费用。形式就是常见的短视频 + 即时编程反馈，但视频制作的精致程度让我觉得钱花的值得。不过网页版视频的速度调节选项一直无效，到现在也没修复。

      Learn to Code by Doing - Code School
      Learn to code in the comfort of your browser with video tutorials, programming challenges, and screencasts.
        codeschool.com

      • Coder Manual

        和CodeSchool类似，也是付费网站，但是一次性付费。独特之处在于有且只有一门课程，完全面向Web前端开发，号称是以找工作为目的的一步到位解决方案。（Becoma a job-ready programmer）

        Coder Manual: Interactive Coding Bootcamp
        Become a job-ready programmer. Our interactive online coding bootcamp covers everything you need to know to get hired as a web developer.
          codermanual.com

        • Free Code Camp

          目前正在学习，和上面的Coder Manual同时进行中。和Coder Manual类似，是以“培养面向工作的前端开发者”为目标的综合型培训网站，有非常完善的路线图，（网站自称）如果完全严格按照路线图来学习的话有超过1000小时的练习时间。没有其他计划的话，无脑跟网练习也是不错的选择。亮点是号称在所有练习完成后可达成“Project Ready”成就，解锁慈善项目，为各类慈善组织做免费开发。个人认为这是一个非常好的尝试，一方面为无力负担开发者费用的慈善组织提供志愿者，另一方面为初学者在面对正式工作挑战前提供真实的开发经历。不过我还没有通关尚未解锁这项成就，不知道实际效果是否有理论上那么好。

           

          PS，他们有自己的Slack工作组。

          PPS，以上一切都是免费的。（Coder Manual我说的就是你）

          Learn to code
          We're a community where you learn to code by helping nonprofits. Build your full stack JavaScript Portfolio today.
            freecodecamp.com

          • 技术类视频课程

            大量视频教程，不带或只带少量即时编程反馈，由学生自己完成知识点评估。类似于传统的纯视频教程。相比于下一类别，这一类网站更偏重于技术应用，或者“How-to”类课程。

          • Udemy

            付费网站，按每门课程单独收费，一次性收费，终身观看。课程数目相当巨大，（1000+）而且不限于计算机类，几乎涵盖生活中各个方面。大部分课程价格贵得离谱，但隔三差五就会抽风一样的全站$19甩卖，甚至$9甩卖。

          • 慕课

            个人认为是国内同类网站中做得最好的一个。

          • Microsoft Virtual Academy

          • Pluralsight

            由于CodeSchool被收购才知道的网站，还没有深入尝试过。

            Pluralsight – Hardcore Developer and IT Training
            Unlimited online developer and IT courses, starting at $29 per month with new training daily. Courses authored by top industry experts on the most in-demand skills.
              pluralsight.com

              最早也是目前最火，并被认为是“血统纯正”的在线教育网站。提供的课程偏重理论和基础知识，更多“Why”的内容。

            • Coursera

            • Treehouse

              很早就听说过的一个技术培训类网站，但因为是收费网站，不敢轻易尝试。经@lqchao007 在评论中推荐，添加进来。

              Learn Web Design, Web Development, and More | Treehouse
                teamtreehouse.com

              • 技术讲座类视频

              • Channel 9

              • Google Developer on Youtube

              • Jetbrains TV on Youtube

              • 在线成就汇总/展示

                比较新的一类网站，着重关注在线教育服务暂时无法完美解决的”对成就和证书提供展示和背书“的问题。暂时只发现了一家，国内似乎没有对应的服务。个人非常看好的方向。

              • Degreed

                可以接入的网站非常之多，常见的如codeschool，codecademy，treehouse，MVA等等都有收录。采用了类游戏化机制，积分和成就系统，连pocket中阅读过的文章也可以攒积分。在接入的网站中也发现了一些以前不知道的美好站点，尝试后会加入到列表中。

                Get Credit For Lifelong Learning - College, MOOCs & More | Degreed
                Degreed 'jailbreaks the degree' by measuring all academic, professional, and lifelong learning. Create your profile to start getting credit for your education.
                  degreed.com
