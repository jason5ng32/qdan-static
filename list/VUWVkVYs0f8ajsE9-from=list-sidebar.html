实用Chrome插件推荐

Google Chrome不仅速度快，而且通过在Chrome Web Store添加扩展能满足各类用户的需求。笔者从09年开始使用Google Chrome，至今也体验过不少好用的插件，下面，笔者和大家分享一些好用的插件，希望大家能从Google Chrome中获得更好的体验。

  1. Infinity新标签页

    自由定制chrome新标签页；扩展还集成天气、备忘录、Todolist等实用工具，绝对是同类应用中的上品。

    Infinity New Tab
      infinitynewtab.com

    • crxMouse Chrome Gestures

      方便,快捷,充分发掘鼠标的所有操作.功能包括:鼠标手势,超级拖曳,滚轮手势,摇杆手势,平滑滚动,标签页列表等.

    • Adblock Plus

      免费的广告拦截器，可阻止所有烦人的广告及恶意软件和跟踪。

    • FaWave(发微)

      支持多微博的同时提示、无缝切换。

    • OneTab

      号称能帮你节省95%的内存开销；还可以对众多标签页进行管理，如果你经常需要打开很多标签页，建议你可以试试这个。

    • Yet another flags

      显示该网站服务器的IP地址。

    • 网页截图

      功能全面的截图工具。

    • Search All

      通过一个简单的点击即可切换不同搜索引擎，而且搜索结果还整合了书签和历史记录，非常实用。

    • Google翻译

      体验很好的翻译工具。

    • SimpleExtManager

      扩展管理工具。

    • Proxy SwitchyOmega

      轻松快捷地管理和切换多个代理设置。

    • Search the current site (站内搜索)

      为您提供当前浏览网站的站内搜索功能。

    • 印象笔记·剪藏

      一键保存精彩网页内容到印象笔记帐户；这类型的工具有很多，但印象笔记的绝对是最好用的！

    • Diigo Web Collector

      在线高亮标识文章，强烈推荐。

    • LastPass

      多平台密码管理应用。

    • 考虑到国内可能访问不了Chrome Web Store，推荐使用此工具下载扩展

    • 也可以到笔者博客私信我获取可用的Google Hosts。
