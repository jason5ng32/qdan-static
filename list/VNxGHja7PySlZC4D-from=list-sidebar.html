高桥名人之冒险岛 II 每关名称

《高桥名人之冒险岛 II》（日版名：高橋名人の冒険島II，英文版名：Adventure Island）是一款横向卷轴平台游戏，游戏由 Now Production 开发，Hudson Soft 于 1991 年发行于 FC 游戏机平台。该游戏为《高桥名人之冒险岛》的续篇，也是冒险岛系列在 FC 平台上的第二个作品。

  1. Fern Island - 蕨类岛

  2. Lake Island - 湖泊岛

  3. Desert Island - 沙漠岛

  4. Ice Island - 冰雪岛

  5. Cave Island - 山洞岛

  6. Cloud Island - 白云岛

  7. Volcano Island - 火山岛

  8. Dinosaur Island - 恐龙岛
