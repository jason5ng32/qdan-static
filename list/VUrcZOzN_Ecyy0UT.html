那些你值得安装的 Gnome 扩展

前些天升级到了 Fedora 22 Beta ，又从 XFCE 改用了 Gnome 。还记得早些时候大家对 Gnome 3 褒贬不一，好在现在越来越易用了，只要你耐心改造一番就能得心应手。

前提：安装了 gnome-tweak-tool 和 Firefox 。

  1. Firefox 最好是通过包管理器安装的

    这样在扩展管理页面中才可以看到 Gnome Shell Integration 这个插件（不然你就要自行安装 libgnome-shell-browser-plugin.so 了）。 之后从 tweak tool 中的 extensions 页管理扩展，也可进入 https://extensions.gnome.org/ 管理。

    GNOME Shell Extensions
      extensions.gnome.org

    • Applications Menu by fmuellner

      将你的左上角的「活动」按钮变成怀旧版「应用程序」菜单，如果你也是比较钟情这种打开应用的方式的话，推荐安装。点击「应用程序」展开菜单，鼠标撞向左上角依然是会展示「活动」的。

      Applications Menu - GNOME Shell Extensions
        extensions.gnome.org

      • Battery Percentage by eagleoneraptor

        右上角电量显示百分比。

        Battery Percentage - GNOME Shell Extensions
          extensions.gnome.org

        • Caffeine by eon

          在右上角显示一个咖啡杯（也可能是茶杯？），点击可切换两种状态：杯子冒热气时禁止熄屏，不冒热气时处于屏保自动的模式。而且当任一应用全屏时会自动禁止熄屏。

           

          因为我不知道在看网页视频时如何让电脑不熄屏，只好用它了。

          Caffeine - GNOME Shell Extensions
            extensions.gnome.org

          • Coverflow Alt-Tab

            Alt+Tab 切换应用时有 3D 效果，如图所示。

            Coverflow Alt-Tab - GNOME Shell Extensions
              extensions.gnome.org

            • Drop Down Terminal by zzrough

              按下左上角的波浪号键，一个终端就会从屏幕上方放下来，像「卷帘门」一样……再按一下收上去，很方便。

              Drop Down Terminal - GNOME Shell Extensions
                extensions.gnome.org

              • Media player indicator by eon

                在右上角放一个音乐播放器的控制栏。

                Media player indicator - GNOME Shell Extensions
                  extensions.gnome.org

                • Places Status Indicator by fmuellner

                  在左上角放上一个复古的「位置」菜单。

                  Places Status Indicator - GNOME Shell Extensions
                    extensions.gnome.org

                  • Simple Dock by optimisme

                    将「活动」页面的「应用图标栏」挪到底部，类似 OS X 的那样。

                    Simple Dock - GNOME Shell Extensions
                      extensions.gnome.org

                    • Suspend Button by laser_b

                      让人想不通的一点就是 Gnome 把「挂起（睡眠）」按钮隐藏起来了，如果你想通过鼠标让电脑睡眠，你需要点右上角，然后按住 Alt 键，才能看到「关机」按钮变成了「挂起」……

                      这个扩展就是要「挂起」默认显示在「关机」旁边。如果你不常用这个的话也就没必要安装了。

                      Suspend Button - GNOME Shell Extensions
                        extensions.gnome.org

                      • Todo.txt by BartL

                        在右上角增加一个可以快速添加「待办事项」的按钮，一个简单的 To-do-list 。

                        Todo.txt - GNOME Shell Extensions
                          extensions.gnome.org

                        • TopIcons by ag

                          Gnome 3.16 将托盘区放到了左下角，一个可收起/展开的图标。这个应用就是要让图标都回到右上角托盘区。（没错，我是一个怀旧的人……）

                          TopIcons - GNOME Shell Extensions
                            extensions.gnome.org

                          • User Themes by fmuellner

                            将上这个扩展，你就可以通过 gnome-tweak-tool 安装桌面主题了。

                            User Themes - GNOME Shell Extensions
                              extensions.gnome.org

                            • Wobbly Windows by djreep81

                              还记得在 Gnome 2 年代赫赫有名的 Compiz 吗？装上这个扩展，再次回味拖动窗口时的那种「果冻感」吧～

                              Wobbly Windows - GNOME Shell Extensions
                                extensions.gnome.org
