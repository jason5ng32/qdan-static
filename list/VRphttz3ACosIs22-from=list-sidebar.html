微软、苹果、谷歌：免费在线办公套件、同步存储

大趋势下，无论是微软、苹果、谷歌都将其办公软件 WEB 化，无论是在 Windows、Mac OS X、Chrome OS、GNU/Linux 的操作系统下，无须安装客户端，不佔用本地端硬盘空间，更无须担心存储备份文档（全部同步自动化）。用户只需要透过浏览器即可免费线上使用各家的办公套件（同步云存储、文档编辑处理、试算表、幻灯片演示、笔记），实现真正的跨平台。

 

各家特色简介：

微软家族套件，适合庞大複杂性的作业。苹果家族套件，适合简约美观性的呈现。谷歌家族套件，适合实时协作性的项目。另外附上俄罗斯大鳄 Mail.Ru 与 韩系小清新 Naver 的小众选择。在各大浏览器上，只要点击以下轻单，即可直接连结到各家网页版软件。

 

推荐浏览器：Windows 用户建议採用 Chrome 浏览器或 Yandex Alpha 浏览器（同时兼容 Chrome 与 Opera 的插件扩展应用生态圈，美观适合外貌控）, Mac OS X 用户建议採用预载的 Safari 浏览器，GNU/Linux 用户建议採用全开源的 Firefox 浏览器，Chrome OS 用户就原生直奔了。以上建议基于『流畅性』的体验标准。

 

推荐移动端：苹果的 iPad 平板电脑外加实体键盘的操作感受与使用情境相当优异，微软、苹果、谷歌的办公软件皆可在应用商店免费安装，提醒：每一个应用将会佔用约数百MB的 iPad 本地空间（皆可说是庞大佔空间的 iOS 应用）。
