如何在 Chrome 上安装 Android 应用

  1. 安装最新的 Chrome (v41)

    需要科学上网

    Chrome Browser
    Google Chrome is a fast, simple, and secure web browser, built for the modern web.
      google.com

    • 从商店安装 ARC Welder app

      大约 100 多兆，需要有耐心

      ARC Welder
      Package Android APKs for ARC (App Runtime for Chrome)
        chrome.google.com

      • 安装完成后可在 chrome://apps 里看到

      • 你还可以将其添加到桌面或菜单中

        在图标上右击，创建快捷方式。单主 Fedora (Xfce) 如图。

      • 启动 ARC Welder app

      • 先选择一个用于存储临时文件的路径

      • 然后添加 APK 文件

        国外的 Android 应用多是发布在 Google Play 商店的，如果你有商店的链接可以通过下面这个网站下载 APK 文件。

        APK Downloader [Latest] Download Directly | Chrome Extension v3 (Evozi Official)
        Download APKs Directly From Google Play To Your Computer With APK Downloader Extension For Google Chrome
          apps.evozi.com

        • 启动 Android 程序

          选择“横、竖屏”，“手机、平板”模式，是否允许复制、粘贴等。

           

          PS 并非所有的 App 都能用。 QQ、微信无法启动，以下几款应用亲测可用。

          歌曲可播放。

        • 意义

        • Linux 用户的福音

          由于用户群体较少，多数桌面应用仅支持 Windows 和 OS X 。通过这一方法，Linux 用户可以借助 Chrome 使用更多应用。

        • 移动应用的跨平台化

          主要是一些通讯工具和游戏，这些应用被引入桌面平台之后，可以提高沟通效率或者更方便用户在工作间隙打发时间。

        • Chrome 的一大步

          Android 充当 PC 系统，跨入桌面领域早已不是新闻了，看好的不少，唱衰的也有。如今 Chrome 又为用户开启了一个新的纪元， Google 通过“左右互搏”之术，正一步步蚕食整个市场。

        • 对于前端开发者
