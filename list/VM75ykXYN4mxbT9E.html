  1. apktool

    解包、打包apk

      code.google.com

    • smali

      反汇编成smali代码，apktool集成了这个软件

      smali - An assembler/disassembler for Android's dex format - Google Project Hosting
        code.google.com

      • APK Studio

        集成了各种常用功能的IDE

        APK Studio | Android Reverse-Engineering
        APK Studio is a cross&#61;platform IDE for decompiling&#47;editing &#38; then recompiling of android application binaries.
          apkstudio.codeplex.com

        • Virtuous Ten Studio

          同上，只是这个功能稍微强大一些，但是操作逻辑复杂一些

          Virtuous Ten Studio
            virtuous-ten-studio.com

          • IDA

            主要用来分析.so文件，但是实际上你也可以直接用它分析apk

            IDA: About
            The IDA Disassembler and debugger is a multi-processor disassembler and debugger hosted on the Windows, Linux and Mac OS X Platforms.
              hex-rays.com

            • jadx

              反编译成java代码，自带GUI

              skylot/jadx
              jadx - Dex to Java decompiler
                github.com

              • JEB Decompiler

                据说很牛逼的反编译器，可惜买不起

                PNF Software - Creator of JEB, the Android app Decompiler
                PNF Software is the maker of JEB, a powerful interactive Android app decompiler, built by and for security engineers.Decompile APK files and obfuscated Dalvik bytecode. Script your automation needs with our Java and Python API.
                  pnfsoftware.com

                • Dare

                  据说还不错的反编译器

                  Dare Homepage
                    siis.cse.psu.edu

                  • Androguard

                    一整套命令行工具

                    androguard/androguard
                    androguard - Reverse engineering, Malware and goodware analysis of Android applications ... and more (ninja !)
                      github.com

                        code.google.com
