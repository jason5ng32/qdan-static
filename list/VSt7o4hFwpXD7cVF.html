美国总统大选4个参选人的网站对比

希拉里·克林顿宣布要参加 2016 年大选，她（的团队）也相应建立了一个拉票网站。

 

我们不妨对比一下泰德·科鲁兹等另外3个参选人，从技术上剖析一下他们的拉票网站布得怎么样。

  1. 四位参选人及网站

  2. 希拉里·克林顿

    以下将其网站简称为 希拉里

  3. 泰德·科鲁兹

    以下将其网站简称为 科鲁兹

  4. 兰德·保罗

    以下将其网站简称为 保罗

  5. 马克罗·鲁比奥

    以下将其网站简称为 鲁比奥

  6. HTTPS

  7. 是否支持 HTTPS

    科鲁兹率先失利（而且还不支持 http 裸域名），其他三个网站都支持 HTTPS

     

    另 https://tedcruz.org 会 404 ，而 https://www.tedcruz.org 会跳转到 http 。

  8. 是否默认 HTTPS

    保罗不是默认，只有希拉里和鲁比奥在竞争了

     

    但是 https://hillaryclinton.com 会自动跳转到 https://www.hillaryclinton.com/

    而加了 www 的鲁比奥也会跳转到裸域名 https://marcorubio.com

  9. 是否支持 HSTS

    希拉里和鲁比奥都不支持，此轮 GAME OVER

     

    PS 网站可以选择使用 HSTS 策略，来让浏览器强制使用 HTTPS 与网站进行通信，以减少会话劫持风险。

      • CDN

        内容分发网络 Content Delivery Network——CDN，简单来说就是把用户需要访问的内容提前拉到用户所在的地方去。CDN 可以实现将源站内容分发至全国所有的节点，缩短用户查看内容的延迟，使内容传输的更快、更稳定。

         

        国内的有腾讯云、阿里云、网宿等。

        • WordPress 4.1.1

          科鲁兹和鲁比奥都是用的 WordPress

        • PHP 5.5.9 Ubuntu

          保罗的后台也是 PHP ，主机是乌班图。

           

          PHP 是最好的编程语言，yeah!

        • Python

          只有希拉里是用的 Python 写的后台，还用到了gunicorn 19.1.1 + Varnish 和 groundwork 。

        • 域名注册

        • 注册商

          希拉里的域名是从 Network Solutions 注册的；

          科鲁兹和鲁比奥都是从 GoDaddy 注册的；

          保罗是从 Fabulous.com 注册的。

          希拉里的域名没有 whois 保护；

          科鲁兹和鲁比奥都是用的 Domains By Proxy® 的 whois 保护；

          保罗的 whois 保护是 Whois Privacy Services Pty Ltd 提供的。
