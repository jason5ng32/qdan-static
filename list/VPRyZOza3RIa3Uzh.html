Moto X测评摩托罗拉手机

Moto X 2014 版测评 : 斑斓的绿松蓝色，老摩粉的挚爱

”据说，无数老摩粉们在“Hello Moto”的铃声响起的时候，湿了眼眶。今天要放出的测评正是Moto X中国版的。”

  1. 火星文：不久前，摩托罗拉携Moto X,Moto G,New Moto X Pro 三款智能手机重返中国。据说，无数老摩粉们在“Hello Moto”的铃声响起的时候，湿了眼眶。今天要放出的测评正是Moto X中国版的。

    Moto X 的入华，是一代老摩粉的期待，更是Moto对于资本市场的一次妥协。同样面对中国大陆这片神奇的土地，google骄傲地选择了放弃中国市场，而Moto不得不进军这一块有可能是世界上最大的智能手机消费市场，即使为了适应中国市场而做出所谓“本土化”的妥协也在所不惜。

  2. 图2

    手中这台是2014版的Moto X也就是第二代Moto X国行版本。颜色为绿松蓝，笔者是天生色弱，对于色彩斑斓的机器总是心怀敬畏….

  3. 图3

    Moto X 作为Moto产品线上的旗舰，硬件配置自然不必说，5.2英寸的AMOLED弧面屏幕，屏幕的色彩，屏幕边缘圆润的手感在Android阵营可谓绝妙。金属边框也打磨的十分讨喜，曾有大神说“直线属于人类，曲线属于上帝”。Moto给顾客上帝准备的这台机器从头到脚充斥着令人喜悦的曲线设计，同时用锯齿和平滑两种表面处理的锁屏键和音量控制键让人避免了误操作。一点点设计上的小改变，让用户体验瞬间上升一个层次。

  4. 图4

    随后国行版本将会推出的Moto maker将会进一步将“选择的权利”交给用户，用户可以自定义手机细节的配色、后盖的材质甚至是系统默认的壁纸。这样大胆的将设计放权给用户不可谓不大胆，同时也是Moto自信的表现，“我们的硬件已经如此出色，搭配上你自己设计的色彩搭配，你还能不喜欢？”

  5. 图5

    私以为语音助手功能一直是智能手机里属于花拳绣腿的功能之一，Moto在这一点上做的也比较保守，不出彩也不算差，测试了“ok Moto x，明早7点叫我起床”之类的指令，识别成功率也不错。但除非真正足够智能，否则语音助手也就只能做做这种事了。

     

    扭动门把手一样扭动两次可以在任意环境下打开摄像头，重复操作一次切换到前置。上下甩动两次打开手电筒，这两个都是异常方便的功能。用过了就会爱上。

     

    令人惋惜的是摄像功能的差劲，无论是对焦速度，夜间成像，都配不上旗舰机的称号，这一点每一个用户都会体会得到，如果你的上一台手机是iPhone，那么Moto X的相机绝对会给你造成心理上的落差，拍照重度用户一定要慎重考虑这个机器。

  6. 图6

    也许是知道自己还是存在致命的短板，所以国行版本在定价上还算厚道，不论是哪个型号的Moto X都将涡轮快速充电器和耳机作为标配送给国内用户，而在美国市场这都是单独购买的，因此国行的盒子都比美版厚实不少，也算是对于老摩粉的一次报答。

     

    2.5GHz的四核处理器加上2G内存足够让Moto X保证系统的流畅性，避免了国内市场安卓机器的普遍尴尬“越用越卡”。上手半个多月，没有遇到一次卡顿，没有重启过一次。

     

    但所谓本土化大家心知肚明，无非就是阉割google服务，这也算是我们市场的一大特色，也因此产生了一系列UI、OS，真说不清楚是好事还是坏事，但目前情况就是这样，矮子里面选将军，下一代Moto X如果能解决拍照这一短板，做中国市场里的将军，完全是有可能的。

     

    如果你是个狂热的搞机者，也许美版的Moto X更加适合你。原汁原味的原生系统比本土化的原生系统更具有可玩儿性；但如果你是个不愿意折腾的用户，同时又厌倦了ios的堆砌app，那么国行Moto X绝对可以让你体验到Android阵营里少有的顺滑体验同时又不失逼格。
