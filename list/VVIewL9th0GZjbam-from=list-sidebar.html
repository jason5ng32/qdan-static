Firefox浏览器扩展一分钟了解轻单精选

一个浏览器重度用户的五星扩展清单之 Firefox 篇

作为一个自认浏览器重度用户以及见习前端开发者，Firefox 和 Chrome 地址栏右边总是一大堆图标。所以来建立两个轻单用来整理一下我认为的好用的扩展，就是这些扩展，让其他浏览器（IE说你呢）相形见绌。个人习惯原因，大部分是工具和效率类的。为了更好地组织条目，先来 Firefox 篇。

  1. Pocket - 体验最好的稍后阅读工具

    我眼中最好的稍后阅读类应用，从ReadItLater时代开始用，当初不愠不火，2012年改名Pocket之后，也浴火重生，全平台。并且专心做好了稍后阅读这一件事，真正的一键添加到未读，最简单也最本质的核心竞争力，正是因为这个原因，抛弃了Evernote的clearly功能。而不选择Instapapaer的原因是，Pocket速度更快，移动端阅读体验也更佳。TIP: Firefox 上用快捷键 Alt + W 添加未读，连鼠标都不用点。

    Pocket
    When you find something you want to view later, put it in Pocket.
      getpocket.com

    • Pushbullet - 推送、共享神器

      必备神器，诸多杀手锏，每一个都戳中需求痛点。必须使用Google或Facebook帐号注册，所以需翻墙，但注册成功之后不翻墙也可以用。

      1. 同步Android 通知（没错，智能手表之流鼓吹的不点亮手机看通知至少在电脑上可以免费解决了）。对于MAC用户，Pushbullet也支持mac app和iOS同步通知。

      2. 直接在浏览器里回复手机短信（也支持whatsapp）

      3. 一键推送当前网页网址到手机，手机上可以直接以通知形式接收到链接（反向也可）

      4. 不需第三方工具实现Universal Clipboard（A处复制，B处粘贴）

      Pushbullet - Your devices working better together
        pushbullet.com

      • VimFx - 脱离鼠标浏览网页

        键盘党福音，脱离鼠标浏览网页。之所以没有使用Vimperator，因为学习曲线太高，并且感觉它对许多原生浏览器环境的破坏太大。常用的姿势有 f键 follow link 正中需求痛点，r键刷新，t打开新标签，x关闭当前标签，大写X 恢复刚刚关闭的标签，d向下滚动， u向上滚动。按 ? 可以看到完整列表

        VimFx
        VimFx introduces Vim-style keyboard shortcuts for browsing and navigation without using the mouse.
          addons.mozilla.org

        • FoxyProxy - 优秀的代理工具

          代理工具，传统的AutoProxy似乎停止更新了，换用FoxyProxy，功能很强大。当然，它并不能帮你达到翻墙的目的，只是帮你设置代理，要翻墙还需要自己找合适的代理服务器。

          FoxyProxy Standard
          FoxyProxy是一个高级的代理管理工具，它完全替代了Firefox有限的代理功能。它提供比SwitchProxy、ProxyButton、 QuickProxy、xyzproxy、ProxyTex、TorButton等等更多的功能。
            addons.mozilla.org

          • Gmail Notifier - Gmail新邮件提醒

            Gmail被墙了，但不影响它依然是最好用的个人邮箱。在互联网世界，一个产品好不好用很多时候不是它一个人的事，而是围绕它产生的“周边”，它的生态。比如这个扩展，功能很简单，就是新邮件提醒，做的不多也不少。当然，前提是要翻墙才有用。

            Gmail Notifier (restartless)
            支持多个标签和帐户的通知工具，适用于 Google Mail (Gmail)
              addons.mozilla.org

            • Adblock Plus - 有效屏蔽广告

              用来屏蔽广告，差不多算上是firefox必装top 3吧。装好之后确认一下订阅过滤规则EasyList China+EasyList是正确的姿势。

              Adblock Plus
              被广告弄得搓火吗？被跟随鼠标的标语弄得烦心吗？被横幅弄得感到厌恶吗？立即安装 Adblock Plus 来重新取得 Internet 的控制权并且改变您浏览网页的方式。 在 http://www.youtube.com/watch?v=oNvb2SjVjjI 上有简短的视频可供参考
                addons.mozilla.org

              • QrCodeR - 以最便捷的方式生成二维码

                一个小巧方便的二维码分享工具，可以在地址栏直接显示当前网页URL对应的二维码图片，也可以通过右键菜单直接把链接、图片或者选中的文本编码为二维码，接下来就是拿起手机扫扫扫了。简单粗暴的分享方式。

                QrCodeR
                在右键菜单中即可将选定部分、链接或图像编码为QR码。
                  addons.mozilla.org

                • Flagfox - 在地址栏显示国旗

                  根据网站IP地址在地址栏图标直接显示国旗，快速判断网站服务器所在的国家。有时候还可以顺便认出来一些欧洲国家的国旗长长姿势。

                  Flagfox
                  显示描述当前服务器位置的国旗。
                    addons.mozilla.org

                  • Wappalyzer - 直观地了解网站使用的技术

                    又一个让你心甘情愿把地址栏让给它占领的好东西，在地址栏显示当前网站相关的技术细节。包括：前端所用的JS框架，脚本工具（例如Google Adsense），有时也可以判断出后端使用的服务器类型，开发语言。技术人员眼中的好工具。

                    Wappalyzer
                    Identifies software on the web
                      wappalyzer.com

                    • FireSSH - 浏览器里也可以SSH

                      没错，浏览器里也可以SSH。一个ssh客户端该有的基本功能它已经都有了，并且中文不乱码，并且支持openssl的私玥文件格式pem。这意味着，在Windows下完全可以替换掉win32 ssh客户端言必称的putty（深深的怨念：如果要用putty连接我的ec2主机，还要手动把亚马逊提供的pem格式的私玥转换成putty支持的ppk！！）当然，Linux下直接在终端里SSH就可以了，并不需要这些劳什子。

                      FireSSH
                      FireSSH is a free, cross-platform SSH terminal client for Mozilla Firefox. Written entirely in Javascript!
                        addons.mozilla.org

                      • FireFTP - 浏览器里还可以FTP

                        一个装在浏览器里的FTP客户端，该有的功能都有了，多余的功能都没有。提供了经典的两栏布局，使用简单方便。

                        FireFTP
                        FireFTP是Mozilla Firefox的一个免费的、安全的、跨平台的FTP/SFTP客户端，它提供了方便而直观地访问FTP/SFTP服务器的功能。
                          addons.mozilla.org

                        • Stylish - 自定义样式

                          自定义CSS样式，可以基于某一个网页，也可以基于某个域名甚至所有页面。要用这个扩展，需要满足两点 1.强迫症，对某些网站的样子或字体或配色不能忍 2.懂点CSS

                          Stylish
                          Restyle the web with Stylish, a user styles manager. Stylish lets you easily install themes and skins for Google, Facebook, YouTube, Orkut, and many, many other sites. You can even customize Firefox and other programs themselves.
                            addons.mozilla.org

                          • Greasemonkey - 自定义脚本

                            自定义js脚本，非常强大，使用js以hacker的方式给别人的网站增加自己的功能。即使不懂js的也可以拿大神写出来的脚本来用，比如这个网站 https://openuserjs.org/

                            Greasemonkey
                            Customize the way a web page displays or behaves, by using small bits of JavaScript.
                              addons.mozilla.org

                            • Firebug - 压轴

                              业界标杆，F12届的教父，拿来压轴，没话说。最后，Firefox还是装Developer Edition的好，暗色主题，酷炫爆棚。

                              PS：我发现，似乎Developer Edition的F12工具与firebug是一样的东西，从Page Inspector到Console都一样，但常规安装firebug还是并不冲突。我在想开发者版中也许事实上内置了firebug的各个组件，但从官方博客里并没有看到这个说法。

                              Firefox Developer Edition
                              为构建网络的人打造，这是唯一一款专门为开发者准备的浏览器。
                                mozilla.org
