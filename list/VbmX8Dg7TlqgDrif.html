数据库

SQL + MySQL + NoSQL + MongoDB + MariaDB + Oracle SQL

  1. 《SQL必知必会》

    作者：Ben Forta

     

    译者：钟鸣 刘晓霞

     

    书号：978-7-115-31398-0

     

    定价：29.00 元

     

    页数：256

     

     

    SQL经典畅销书，内容丰富，简洁实用

     

    麻省理工学院、伊利诺伊大学等众多大学的参考教材

     

     

    本书由浅入深地讲解了SQL的基本概念和语法，涉及数据的排序、过滤和分组，以及表、视图、联结、子查询、游标、存储过程和触发器等内容，实例丰富，便于查阅。新版增加了针对Apache Open Office Base、MariaDB、SQLite等DBMS的描述，并根据最新版本的Oracle、SQL Server、MySQL和PostgreSQL更新了相关示例。

  2. 《MySQL必知必会》

    作者：Ben Forta

     

    译者：刘晓霞 钟鸣

     

    书号：978-7-115-19112-0/TP

     

    定价：39.00 元

     

    页数：256

     

     

    各大网店最畅销MySQL图书

     

    世界知名技术作家Ben Forta力作

     

    紧贴实战，全面覆盖所有内容

     

     

    本书是经典畅销书《SQL必知必会》之后，作者应众多读者的请求编写的，专门针对MySQL用户。书中继承了《SQL必知必会》的优点，没有过多阐述数据库基础理论，而是紧贴实战需要，直接从数据检索开始，逐步深入各种复杂的内容，包括联结的使用、子查询、正则表达式和基于全文本的搜索、存储过程、游标、触发器、表约束，等等。通过本书，读者能够掌握扎实的基本功，迅速成为MySQL高手。

  3. 《NoSQL数据库入门》

    作者：佐佐木达也

     

    译者：罗勇

     

    书号：978-7-115-27950-7

     

    定价：45.00 元

     

    页数：199

     

     

    介绍memcached、Tokyo Tyrant、Redis、MongoDB

     

    如何基于MySQL应用NoSQL技术特性

     

     

    本书详细地介绍了NoSQL数据库（非关系型数据库）的种类、用途以及使用方法,并对memcached、TokyoTyrant、Redis、MongoDB这4种代表性的NoSQL数据库的特征、适用范围、实现代码进行了深入探讨,并比较了它们的性能。

  4. 《MongoDB权威指南（第2版）》

    作者：Kristina Chodorow

     

    译者：邓强 王明辉

     

    书号：978-7-115-34108-2

     

    定价：79.00 元

     

    页数：412

     

     

    第1版广受好评，新版本在此基础上大幅扩充，内容更加全面、深入

     

    MongoDB项目组成员倾力打造

     

    基础知识、进阶内容、高级管理任务一应俱全

     

     

    本书是一本广受好评的MongoDB权威著作。书中介绍了面向文档的存储方式及利用MongoDB的无模式数据模型处理文档、集合和多个数据库，讲述了如何执行基本的写操作以及各种复杂的条件查询，还介绍了索引、聚合工具以及其他高级查询技术，另外对监控、安全性和身份验证、备份和修复、水平扩展MongoDB数据库等内容也有所涉及。

  5. 《MariaDB原理与实现》

    作者：张金鹏 张成远 季锡强　

     

    书号：978-7-115-38517-8

     

    定价：59.00 元

     

    页数：300

     

     

    京东云数据库组核心成员、资深研发工程师力作

     

    第一本深入剖析MariaDB核心机制的书

     

     

    本书由浅入深地剖析了MariaDB，首先简要介绍了一些基础知识、新特性、对MySQL原有功能所做的扩展以及源代码，接着介绍了底层数据结构、线程池技术、binlog、复制等内容，最后介绍了分布式数据库和京东的分布式数据库系统。

  6. 《精通Oracle SQL（第2版）》

    作者：Karen Morton等

     

    译者：朱浩波

     

    书号：978-7-115-35166-1

     

    定价：99.00 元

     

    页数：540

     

     

    Oracle ACE与OakTable团队专家力作

     

    实用典范，可操作性强

     

    畅销好书全面升级，深入剖析12c

     

     

    《精通Oracle SQL（第2版）》由Oracle数据库领域顶尖技术组织OakTable成员倾力打造，第1版热卖，读者好评如潮。书中主要介绍了如何掌握在Oracle数据库中实现的强大SQL特性，所涵盖的内容涉及SQL核心、SQL执行、分析函数、联结、测试与质量保证等，并提供了大量实用的建议，总结出方方面面的“技巧”，帮助读者快速消化重要知识点。

  7. 《Oracle 12c数据库管理》

    作者：Darl Kuhn

     

    译者：苏宝龙

     

    书号：978-7-115-35540-9

     

    定价：119

     

    页数：672

     

     

    全面详尽，重点突出

     

    内容易懂、示例丰富

     

    基础内容与高级应用相互渗透

     

     

    资深DBA Darl Kuhn在这本书里总结了自己十余年的Oracle实践经验，透彻展示了DBA管理多面数据库环境的方法。书中没有介绍最复杂最尖端的数据库管理技术，而是争取在足以解决任何等级的混乱和复杂问题的前提下，使技术尽可能简单。

  8. 《性能优化》

    作者：Sam Alapati等

     

    译者：朱浩波

     

    书号：978-7-115-37363-2

     

    定价：99.00 元

     

    页数：536

     

     

    世界顶尖Oracle ACE三十余年经验心得

     

    围绕性能调优主线，介绍常见Oracle数据库性能问题

     

     

    性能优化是每个DBA说不出的痛，也是难点。本书针对此现状，围绕性能调优这一主线，由浅入深地介绍了常见的Oracle数据库性能问题。每节分析一个具体的性能问题，并全部采用“问题描述”—“解决方案”—“工作原理”这种独特的介绍风格，针对性强，方便读者理解和查阅。
