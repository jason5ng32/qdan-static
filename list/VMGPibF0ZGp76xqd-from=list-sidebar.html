在 Mac 上写一个简单的应用（with nw.js）

上周我买了个苹果电脑，昨天我写了个苹果应用。

 

You can check my app out in: https://github.com/bcho/enigma

  1. 安装 Homebrew

    One package manager to rule them all!

    Homebrew — The missing package manager for OS X
      brew.sh

    • 使用 Homebrew 安装 Node.js

      $ brew install node

      Homebrew Formulas - Node
      Homebrew (The missing package manager for OS X) formula list to search and discover new formulas
        brewformulas.org

      • 使用 Homebrew 安装你最喜欢的编辑器

        Cool kids use vim.

        Homebrew Formulas - Macvim
        Homebrew (The missing package manager for OS X) formula list to search and discover new formulas
          brewformulas.org

        • 初始化你的工作目录

          $ npm init

          package.json | npm Documentation
          The place where all things npm are documented
            docs.npmjs.com

          • 使用你最喜欢的编辑器写你最棒的代码

            When you writing js, do as the Frontend programmers do.

            JSHint, a JavaScript Code Quality Tool
            JSHint, a JavaScript Code Quality Tool
              jshint.com

            • 安装 node-webkit-builder 到你的包依赖中

              $ npm install node-webkit-builder --save-dev

              npm: node-webkit-builder
              node-webkit-builder
                npmjs.com

              • 使用 node-webkit-builder 来试运行你的应用

                $ ./node_modules/node-webkit-builder/bin/nwbuild -r . # Hint: you may want to use python-livereload to help developing on the fly.

                lepture/python-livereload
                python-livereload - livereload server in python
                  github.com

                • Feeling good? 发布你的应用

                  $ ./node_modules/node-webkit-builder/bin/nwbuild -o build .

                  nwjs/nw.js
                  nw.js - Call all Node.js modules directly from DOM and enable a new way of writing applications with all Web technologies.
                    github.com

                  • 将你的应用拖到 Application 文件夹

                    You may want to update icon, description etc, don't forget to checkout nw.js's wiki.

                    nwjs/nw.js
                    nw.js - Call all Node.js modules directly from DOM and enable a new way of writing applications with all Web technologies.
