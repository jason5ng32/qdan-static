Firefox 大家庭的成员关系

Mozilla 一直是自由软件的先锋，其旗下的 Firefox 浏览器也深受广大用户的喜爱。由于 Firefox 的开源属性，广大基于 Firefox 的派生版“兄弟”也繁衍生息，以满足不同需求的用户。

 

本轻单带你盘点一下 Firefox 大家庭的成员关系。

  1. Firefox 官方系列

    Firefox 其实不是“火狐”，而是“小熊貓”（Red Panda）的昵称。小熊貓是一种红色软毛，与浣熊科和鼬科相近，濒临灭绝的哺乳动物。

  2. 每夜版（Nightly）

    汇集每天源代码的改动，提供最新的尚未被Mozilla质量保证团队测试的新功能，不保证稳定性。

     

    支持 Win 64 bit ，而且 Linux 版只支持 Intel CPU 。

    Firefox Nightly Builds
      nightly.mozilla.org

    • 曙光版（Aurora）已更名为 Developer Edition

      同样每天更新，每六周合并一次来自Nightly的最新代码，它将修复在Nightly中出现的关键问题，能在基本稳定的前提下提供新功能，自35版起，被更名为Developer Edition。

       

      2015年3月3日， Mozilla 推出了首款面向 Windows 平台的 64 位浏览器 Firefox Developer Edition 38 。

      Mozilla Firefox Web Browser — Download Firefox Developer Edition or Beta & Help Determine the Next Firefox
      Download and test Firefox future releases. Experience the newest innovations with the Developer Edition or Beta builds.
        mozilla.org

      • 测试版（Beta）

        会不定期快速更新，用以进一步测试其稳定性。从Beta开始，Mozilla会提供包含数字签名的安装包及可执行程序，并使用正式的Firefox名称及标志。

        Mozilla Firefox Web Browser — Download Firefox Developer Edition or Beta & Help Determine the Next Firefox
        Download and test Firefox future releases. Experience the newest innovations with the Developer Edition or Beta builds.
          mozilla.org

        • 正式版（Release）

          最终稳定发布版本，每六星期发布一版。

          Choose the independent browser
          Firefox is non-profit, non-corporate, non-compromised. Choosing Firefox isn’t just choosing a browser. It’s a vote for personal freedom.
            mozilla.org

          • 长期支持版本（Extended Support Release，简称“ESR”）

            向不能或不愿每6周升级一次的用户提供的版本，其每42周更新一次。

             

            Tor Browser 即基于 ESR 。

            Index of /pub/mozilla.org/firefox/releases/latest-esr
              ftp.mozilla.org

            • Waterfox

              Waterfox is a high performance browser based on the Mozilla platform. Made specifically for 64-Bit systems, Waterfox has one thing in mind: speed.

               

              基于 Mozilla 二次开发，强调 64 位，速度！支持 Win 和 OS X 。

              Waterfox - The fastest 64-Bit browser on the web
              Waterfox is the highest performing browser based on the Mozilla platform. Made specifically for 64-Bit systems, Waterfox has one thing in mind: speed.
                waterfoxproject.org

              • Pale Moon

                是一个基于Firefox浏览器修改优化而来的衍生版本，在Pale Moon项目的官方主页上，项目组人员详细的描述了Pale Moon相比于Firefox原版的修改内容，涉及针对各种处理器的特殊优化，以及对于内部脚本代码的修改和调整。

                The Pale Moon Project homepage
                Pale Moon is an Open Source, Firefox-based web browser available for Microsoft Windows and Linux, focusing on efficiency and ease of use.
                  palemoon.org

                • GNU IceCat

                  旧称 GNU IceWeasel ，是 Mozilla Firefox 的一个分支，由 GNU 计划进行再发布。目前只支持i386架构。

                   

                  Tails OS 中的 Tor Browser 便是基于 Iceweasel 的。

                  GNUzilla and IceCat - GNU Project - Free Software Foundation
                    gnu.org

                  • SeaMonkey

                    前身是 Mozilla Application Suite ，是一个跨平台的互联网套件，包括浏览器、邮件客户端、网页开发软件及 IRC 客户端。 Mozilla Application Suite 由网景发起，2003 年至 2006 年由 Mozilla 基金会维护。Mozilla Suite 后来被 SeaMonkey 取代，并由社区维护。

                     

                    可在 Linux、Windows、Mac OS X 下运行。

                    The SeaMonkey® Project
                      seamonkey-project.org

                    • Thunderbird

                      从 Mozilla Application Suite 独立出来的邮件、RSS、IRC 客户端软件。 Add-on 丰富，强大，彪悍。

                      Mozilla Thunderbird
                      Mozilla Thunderbird is created by a global non-profit dedicated to putting individuals in control and shaping the future of the web for the public good.
                        mozilla.org
