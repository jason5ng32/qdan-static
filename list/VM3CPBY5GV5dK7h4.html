Chrome扩展个人推荐

我自己平常使用的Chrome扩展推荐给大家，只适合像我这样的懒人使用==

（现在Google的产品包括Webstore都无法正常登陆了，当然这难不倒大家的对不对~）

ps.像Gmail、Wunderlist这类的应用入口类扩展就不推荐了

  1. 微度

    标签页应用，可保存最常用的网站入口，并且可登陆账号同步，非常方便（话说是不是把我的常用网站给暴露了……）

  2. 今日头条

    阅读新闻或者感兴趣资讯的最佳app，任何时间拿出来刷一刷，总能有不错的内容

    《今日头条》你关心的，才是头条！ - 头条网 TouTiao.com
      toutiao.com

    • Momentum

      相当文艺的新标签页扩展，美图、美句、天气、to do list、links、search（谷歌）一应俱全，要不是因为links点起来太麻烦（主要是我懒……），早就替代微度了。

      Momentum
      Replace new tab page with a personal dashboard featuring todo, weather, and inspiration.
        chrome.google.com

      • Shortcut Manager

        超级强大的自定义快捷键扩展，支持Tab Action、Page Action等几乎所有的操作的快捷键自定义。配合鼠标手势软件StrokesPlus超级方便，鼠标左滑一下当前页面左边的标签页全都关闭简直不能更爽。

        Shortcut Manager
        Customize shortcut keys; Assign any Javascript code or browser actions to any key strokes.
          chrome.google.com

        • Yet Another Drag and Go

          之前推荐的crxMouse Chrome Gestures被网友JayXon指出有恶意脚本，我自己查了下其要求的权限确实值得怀疑，另外返利链接貌似也存在，为了安全起见就不推荐给大家了。

          这里推荐JayXon说的Yet Another Drag and Go.

          Yet Another Drag and Go
          Drag URL to open at Back/Front, Before/After; Drag text to 4 directions to search by using different search-engines.
            chrome.google.com

          • Better History

            甩Chrome自带历史记录一条街，我已经完全忘记原来的历史记录长啥样子了。

            Better History
            A better look at your browsing history. The best searching, the sharpest interface, and the most useful filters - for your history.
              chrome.google.com

            • 印象笔记·剪藏

              剪藏这个名字起得特别好，将网页像剪报一样地剪下来并收藏起来，然后配以印象笔记强大的同步管理，实在不能更好用。

              Evernote Web Clipper
              Use the Evernote extension to save things you see on the web into your Evernote account.
                chrome.google.com

              • 印象笔记·悦读

                印象笔记出品的又一款良心扩展，阅读一个排版糟糕网站内容的绝好工具，当然也能同步到印象笔记。

                Clearly
                Clearly makes blog posts, articles and webpages clean and easy to read. Save them to Evernote to read anywhere.
                  chrome.google.com

                • 眼不见心不烦

                  微博插件

                  虽然现在微博有点式微，但仍是个获取信息的好渠道，这款扩展可以屏蔽更改你不想看到的元素，还你一个清爽的界面。

                  眼不见心不烦（新浪微博）
                  新浪微博（weibo.com）非官方功能增强插件，可以无限制地屏蔽关键词、用户、来源，去除页面广告和推广微博，反刷屏，还您一个清爽干净的微博！
                    chrome.google.com

                  • 微盘小住手

                    移除微盘的下载并转发按钮;添加一键保存图片到微博功能

                    如果你不想重蹈小沈阳的悲剧的话，就用此插件吧……

                    微盘小住手
                    移除微盘的下载并转发按钮;添加一键保存图片到微博功能
                      chrome.google.com

                    • 小乐图客(ZIG)

                      主要用它的网页图片批量下载（支持批量重命名等操作）和网页截图，美图拼图什么的……我还是用PS吧……

                      Bulk Download Images(ZIG)
                      zzllrr Imager Geek(ZIG) = Image Tool 4 in 1 = Image Downloader(Bulk Download Pictures) + Screen Capturer + Photo Editor & Tiler
                        chrome.google.com

                      • Adblock Plus

                        广告屏蔽类扩展有很多，这是我用的那款。

                        Adblock Plus
                        Used by over 50 million people, a free ad blocker for Chrome that blocks ALL annoying ads, malware and tracking.
                          chrome.google.com

                        • 人人网改造器

                          这个，怎么说呢，它的描述是让人人网用起来舒服一点，可人人网已经作到我基本不用了……仅推荐给还坚守在人人网的童鞋吧……

                          人人网改造器
                          让人人网（renren.com）用起来舒服一点
                            chrome.google.com

                          • WhatFont

                            如描述所言，The easiest way to identify fonts on web pages.（当然了，主要是English fonts）

                            WhatFont
                            The easiest way to identify fonts on web pages.
                              chrome.google.com

                            • 花瓣网页收藏工具

                              作为本土化的Pinterest，花瓣用来收集整理图片确实很顺手，浏览Dribbble和Behance时用这个扩展相当方便。

                              花瓣网页收藏工具
                              这个扩展程序可以让你方便地从任意网站上采集喜欢的图片、视频或者网页截图到花瓣网(huaban.com)上。这是花瓣网官方(huaban.com)提供的扩展。
                                chrome.google.com

                              • Tabs Outliner

                                功能强大的多标签管理器，但由于功能太强大界面太复杂导致懒散的我不怎么用它……

                                Tabs Outliner
                                The Next Generation Session Manager; A Really Working Too Many Open Tabs Solution; And Your Browsing Notebook.
                                  chrome.google.com
