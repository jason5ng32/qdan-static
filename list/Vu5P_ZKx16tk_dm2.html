一分钟了解 2016 年程序员调查报告（Stack Overflow）

Stack Overflow 在过去一年统计了超过 50,000 份程序员的调查问卷，新鲜出炉了最新的数据报告。

  1. 经常访问 StackOverflow 的程序员分布

    从图中可见语言差异还是较大的障碍。

  2. 关于程序员

  3. 开发者中（认为自己是）全栈工程师居多（28.0%）

  4. 全栈工程师中最受欢迎的技术是 JavaScript（85.3%）

  5. 移动端开发中最受欢迎的技术是 Android（61.9%）

  6. 相对于「程序员」和「工程师」，大部分人（71.6%）觉得自己是「开发者」

    当然也有少部分人觉得自己不一样，比如应该是黑客（Hacker 11.4%）、专家（Expert 11.2%）、忍者（Ninja 9.8%）、管理员（Manager 9.2%）、摇滚明星（Rockstar 7.4%）或者大师（Guru 5.0%）。

  7. 年龄介于 25-29 岁之间的最多（28.4%）

    平均年龄是 29.6 岁。

  8. 程序员中九成是男性

  9. 关于技术

  10. 最受常用的语言依然是 JavaScript （55.4%）

    随着 Node 和 Angular 的发展，PHP 受欢迎程度有所下滑。

  11. 最爱的语言是 Rust （79.1%）Swift 次之（72.1％）

  12. 最怕的技术是 VB （79.5%）WordPress 次之（74.3%）

  13. 最想学的是 Android （15.8%） Node.js 次之（14.8%）

  14. 最受欢迎的开发环境 Notepad++ 和 Visual Studio 并列第一（35.6%）

  15. OS X 超越 Win7 成为最受欢迎的操作系统 （26.2%）

  16. 关于工作

  17. 工作中最大的难题是：不切实际的预期、差劲的文档和需求不明

  18. 决策岗位（VP、CTO、CIO）薪酬最高

  19. 公司员工人数多在 20 - 99 人之间（22.0%）

  20. 远程工作依然是少数

  21. 更多信息
