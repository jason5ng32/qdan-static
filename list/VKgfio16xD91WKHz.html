OS X科技史苹果公司插画

可爱插画帮你了解 Mac OS X 的发展史

来源 git-tower.com｜翻译 ourmacs

 

早期 OS X 操作系统在苹果内部以大型猫科动物为代号，之后就干脆直接以这些动物代号做为对外商标，成为各系统版本的简称，直到这一两年来才有所改变。

 

今天咱们特别为大家整理、介绍 OS X 发展史，帮大家温故知新一下。

  1. 2001/3 OS X 10.0 CHEETAH 印度豹

    其实……OS X 操作系统的是从一只熊 (躲在豹后方) 开始，而不是一只猫。(笑)

     

    2000 年 9 月苹果推出测试版操作系统科迪亚克棕熊 Kodiak。当时只要美金 30 元 (约台币 940 元)，就能买到 beta 版 Mac OS X。6 个月后，正式版 OS 10.0 印度豹才正式上市，开启了以猫科动物系列为代号的命名史。OS X 10.0 除了拥有全新的用户界面 Aqua 外，还包括 TextDeit、Preview、Mail 和 QuickTime 等应用程序。系统规格要求则为 128MB RAM 和 800MB 的硬盘空间，以当时来说，算是还蛮高的规格要求。

  2. 2001/9 OS X 10.1 PUMA 美洲狮

    OS X 10.1 上市不到半年，苹果旋即推出美洲狮 OS X 10.1。OS X 10.1 并没有新增许多新的功能，而是聚焦于改善系统表现。从 Mac OS 10.1.2 开始，苹果出厂的计算机也开始以 OS X 系统作为内设操作系统，永久取代 OS 9。

  3. 2002/8 OS X 10.2 JAGUAR 美洲豹

    OS X 10.2 新增 iChat、Address Book 等应用程序，并推出 Universal Access，使得 OS X 操作系统更便于视障、听障或身障人士使用。除此之外，OS X 10.2 还有了新的起始画面，过往 Happy Mac 的标志不再出现，取而代之的是新的苹果标志，也就是如今我们所熟知的那颗被偷咬了一口的苹果。

  4. 2003/10 OS X 10.3 PANTHER 黑豹

    微软与苹果的瑜亮情仇从这时起，又变得更难分难解吗？一方面，苹果以 Safari 做为新的内设网络浏览器，取代 IE for Mac。另一方面，OS X 10.3 在与微软之间的互用性又更臻完善。另一个新功能则是 Expose，可以帮助用户同时管理大量窗口和应用程序。

  5. 2005/4 OS X 10.4 TIGER 老虎

    OS X 10.4 加入了 Spotlight 搜寻功能，可以快速、准确搜寻计算机内信息。并在新的工具栏表 Dashboard 中，纳入计算器、日历、时钟等工具集。此外，OS X 10.4 也开始搭配在不同的硬件装置上，包括第一个配备 Intel 处理器的麦金塔计算机，以及第一代 Apple TV。然而 Apple TV 一直到 2007 年才正式与大家见面。

  6. 2007/10 OS X 10.5 LEOPARD 豹

    OS X 10.4 上市以后，苹果一直忙于开发、推出新的 iOS 系统和 iPhone，所以一直到 2007 年 10 月，新的 OS X 操作系统 OS X 10.5 才面世，算是相隔最久的发行期。OS X 10.5 的特色包括：内建备份系统 Time Machine，支持 64 位 GUI 应用程序，以及配备 Boot Camp，使得麦金塔计算机得以运行包括 Windows 在内等其他作计算机业系统。

  7. 2009/8 OS X 10.6 SNOW LEOPARD 雪豹

    就像雪豹的名字，只比豹多了个雪字，OS X 10.6 也是以 OS X 10.5 的版本为基础跟进开发，不过 OS X 10.6 还跑去向 iOS 偷师，引进 Mac App Store。

     

    另外，OS X 10.6 的推出，也宣告 PowerPC 处理器时代的结束，从此以后，OS X 操作系统只支持搭载 Intel 处理器的麦金塔计算机。

  8. 2011/7 OS X 10.7 LION 狮子

    身为万兽之王的狮子，OS X 10.7 是首款不发行光盘的 Mac OS 系统，改自 Mac App Store 购买下载。OS X 10.7 搭载许多来自于 iOS 的新功能，包括：Gestures、Launchpad、窗口恢复等等。而前阵子风波不断的 iCloud，也是在 OS X 10.7 中首度亮相。

  9. 2012/7 OS X 10.8 MOUNTAIN LION 美洲狮

    OS X 10.8 引进更多 iOS 功能，包括以新的应用程序 Messages 取代 iChat，Reminders、Notification Center。在推出 OS X 10.8 之前，苹果还做了重大决定，之后每年都会推出新一代麦金塔操作系统。

  10. 2013/10 OS X 10.9 MAVERICKS

    OS X 10.9 算是苹果的突破之作。首先，Mavericks 不是动物，而是加州某个冲浪景点，因此 OS X 10.9 的出现，宣示苹果不再以动物命名操作系统，改以加州地名做为命名。另一个重大变革则是苹果宣布用户可以免费升级 OS X 10.9，当然，未来的更新也是免费的。不过免费归免费，OS X 10.9 仍推出了新的应用程序，像是 Maps 和 iBooks 等等。

  11. 2014/10 OS X 10.10 YOSEMITE 优胜美地

    OS X 10.10 以优胜美地为名。它算是麦金塔计算机操作系统历年来变化最大的操作系统，包括趋于扁平化的接口风格，以及半透明的窗口导航字段效果。此外，有了新的 Handoff 功能，用户在各个装置之间操作都能无缝接轨。另一个不同点则是全屏幕操作模式，只要按下位于窗口左上角的绿色灯号，就会启动全屏幕模式。
