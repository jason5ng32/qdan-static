那些近期涌现的设计工具

2015.1.19 新增：[开发] Blocs - Build beautiful websites without touching a line of code

2015.1.15 新增：[效率] Markly - Create Beautiful Specs With Ease

 

---

近两年涌现的各种设计工具，有帮助设计师介入代码阶段的，也有让设计师“失业”的。

大致分为以下几类：

 

1. 原型 - 特指快速生成原型的工具，与生成精细动效的工具区分

2. 动效 - 生成高保真交互、动效的工具

3. 开发 - 产出物可以直接用于 Production。强调：“直接设计，无需写代码”。其中 Weebly 更是将网站上线、域名绑定、电商、论坛等全套服务包揽了，是一个端到端的建站工具而不仅限于设计。

4. 效率 - 工作流、版本管理、效果预览、生成 Spec，都算作这一类。此类下有很多 Sketch，Photoshop 插件，在此不做收录。

5. 灵感 - 收集整理设计灵感。个人感言：现在还没有满意的，还是用 Pinterest 吧。

6. 设计 - 是的，除了 Sketch 还有各种新生设计工具对 Photoshop 虎视眈眈。有专业级的 Skala（未上线）， Affinity Designer；有 Canva，Makr 等让非专业人士上手设计的半自助工具；也有Monodraw 这种充满 geek 气息的奇趣产品。
