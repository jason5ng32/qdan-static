Windows下那些实用的软件

在Windows下十分实用的软件，有逼格、高效率的软件，软件重度使用者必备。

  1. f.lux

    根据你当前时区，自动判断日落日出时间，适当调整屏幕色温，在晚上屏幕会自动慢慢调整成泛黄，从而达到保护视力的目地。

    适合长时间在电脑前工作的必备软件。

    不适合平面设计师等对颜色要高要求的工作人员。

    f.lux
    Software to warm up your computer display at night, to match your indoor lighting.
      justgetflux.com

    • launchy

      键盘类启动软件，在任何地方按下快捷键就弹出软件，输入你要打开的软件名称后，按回车键即可启动这个软件。十分适合那些在电脑上安装了许多软件，但是又想保持桌面整洁的人。

      launchy还支持插件，例如网页搜索插件就可以让你直接输入关键词，回车后直接跳转到浏览器搜索结果页。但是launchy插件不多。

      有一个类似的国产开源软件wox，试用过一段时间，还不太稳定，但它的插件相对比较多。

      Launchy: The Open Source Keystroke Launcher
        launchy.net

      • everything

        代替Windows自带搜索的软件，搜索速度相当快。

        有一个技巧就是，在everything选项里设置一个显示快捷键，以后需要搜索时，直接按快捷键，然后输入关键词按回车键即可。

        Everything Search Engine
          voidtools.com

        • clover

          Clover 是 Windows Explorer 资源管理器的一个扩展，为其增加类似谷歌 Chrome 浏览器的多标签页功能。仅此而已，但十分方便。

          Clover给资源管理器加上类似Chrome的标签 | 易捷科技 EJIE Technology
          Clover Brings Chrome-Style Tabs to Windows Explorer.
            cn.ejie.me

          • Total Commander

            这是一个文件资源管理神器，速度十分快，功能又强大、高效方便、稳定可靠、扩展性与可配置性强。

            当然了，其实大多数人平时可能只用到这个软件10%左右的功能，另外就是此软件入门也相对有点难度。

            善用佳软中有对此软件详细的介绍，写得十分详细。

            Total Commander - home
            Homepage of Total Commander, a file manager replacement for Windows 95/98/NT/2000/XP/Vista/7
              ghisler.com

            • 广告屏蔽大师

              屏蔽广告软件，可以完全屏蔽网页、软件内的广告，可以自定义规则。

              同类软件中的奶牛也不错，国产的adsafe则慢慢地变质了。

              广告屏蔽大师
              广告拦截大师是一款优秀无敌的去广告软件，能全面过滤各大网站弹窗、视频片头等广告。
                adbyby.com

              • cmder

                这个软件可能普通人较少用得上，但IT相关的工作者（特别是程序员）就特常用了。

                cmder 是为 Windows 提供的一个便携式控制台仿真器。支持许多Linux下的命令。cmder是一个绿色软件，还为mini版和full版，full版包含了msysgit

                cmder | Console Emulator
                cmder is software package that provides great console experience even on Windows
                  gooseberrycreative.com

                • 柠檬桌面

                  快速整理桌面图标，自动将桌面上的图标按分类归纳入图标盒子里。例如：PS、Flash等图标它会自动归纳到“设计开发”图标盒子，QQ、旺旺等自动归纳到“社交聊天”图标盒子，桌面上的文件归纳到“文件”图标盒子。

                  柠檬桌面官方网站首页
                  柠檬桌面官方网站，轻巧方便好用的桌面管理工具
                    smartdesktop.sinaapp.com
