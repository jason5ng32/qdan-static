App易用性效率应用推荐Windows（视窗操作系统）

小众却能大幅提升Windows易用性的免费APP

推荐一些众所“不”周知的轻量级APP，都是我经常使用的，善用它们可大幅提升Windows的易用性。

这些APP具有共同的特点：免费、绿色、轻量级。

  1. Windows三大神器之一：Listary

    国人开发，之前促销时入的，Windows文件管理利器，在某种程度上可以替代Total Commander。虽然有Pro版，但免费版完全够用。

    Listary - Fabulous search utility | Keep files at your fingertips
    Listary is a unique search utility for Windows. All you have to do is just type the file name that you’re looking for, and Listary will display the search
      listary.com

    • Windows三大神器之二：Everthing

      文件搜索利器，速度超快！

      Everything Search Engine
        voidtools.com

      • Windows三大神器之三：Q-Dir

        第三方资源管理器非TC莫属，但大多数用户只使用了50%左右的功能，仅仅只用来管理文件的话，Q-Dir更能胜任，它具有：多栏式窗口，多Tab等方便目录切换、文件拷贝的场合。

        Q-Dir the Quad Explorer for Windows - Q-Dir
        Q-Dir is a Windows Quad Explorer and makes your files and folder easy to manage. Fast and easy access, with an amazing Quadruped-View technique.
          softwareok.com

        • 拥有如MAC般丝滑字体：MacType

          Windows最为诟病的问题之一：字体不好看，MacType恰恰解决了这个问题。

          MacType - 维基百科，自由的百科全书
            zh.wikipedia.org

          • 来自岛国的启动器：CLaunch

            作为启动类的APP，可以推荐的很多，但我经常使用的启动器确是一个非常小众的产品：CLaunch。

            如果说其他启动器都是以键盘操作为主，那么CLaunch则是以鼠标操作为主。当双手离开键盘时（如看视频，单纯的浏览文档等单手操作鼠标的场景），那么，只要按住鼠标右键（全局可用，多种预设场景）即可启动它。

            CLaunch
              hp.vector.co.jp

            • 高效的Windows窗口切换器：VistaSwitcher（替代Alt + Tab）

              小众推荐的APP，完美替代Alt + Tab。

              VistaSwitcher - Manage Windows and Switch Tasks Easier with VistaSwitcher (Alt-Tab Replacement) - NTWind Software
                ntwind.com

              • 让Caps Lock能像MAC那样显示：无线键盘指示灯（Wireless Keyboard Indicator）

                在切换大小写时，有MAC的视觉体验...（MAC党必备）

                官网地址已经不好使了，随便G一下很多都提供下载。

              • 增强型的剪切板管理器：Ditto

                免费、开源的剪贴板增强软件，支持64位操作系统，支持中文，还能在局域网中使用，这就够了，Ctrl + C/V 党必备。

                Ditto clipboard managerDitto clipboard managerDitto clipboard manager
                  ditto-cp.sourceforge.net

                • 让你的鼠标活起来：MouseInc

                  鼠标增强类APP，我最经常使用的功能：滚轮穿透。虽然能实现这个功能的APP很多，但MouseInc算是最小巧，绿色的一款了。MAC党还可以使用它的 “滚轮自然滚动” 功能。

                  MouseInc v1.10 | 耍下
                  MouseInc 是一款鼠标增强软件更新日志：　　1、修复部分bug，比如依赖运行库。　　2、增加居中窗口功能。　　3、增加同步滚动窗口功能。
                    shuax.com

                  • 任务栏时间显示更有用处：CHKen Tray Clock（ 增强型的任务栏时间）

                    小众上介绍的APP，任务栏时间增强工具。可以显示农历，星期，甚至还可以定时提醒，目前网络状况以及 CPU 使用情况，相当于Windows版的iStat Menus。

                    CHKen Tray Clock – 任务栏时间增强工具
                    CHKen Tray Clock 是一款任务栏时间增强工具， 可以显示农历，星期，甚至还可以定时提醒，目前网络状况以及 CPU 使用情况。CHKen Player 出品。......
                      appinn.com

                    • 电脑当前状态一栏无余：软媒魔方雷达

                      虽然软媒魔方显得不那么Geek和小众，但有些功能还是非常不错的，比如雷达，虽然类似的APP很多，但大多比较“流氓”，至少软媒魔方提供了绿色版安装，显得大度！

                      结合CHKen Tray Clock基本上可以代替iStat Menus了，MAC党必备。

                      软媒魔方官网 - 新一代优化大师，支持Win10、Win8.1、Win7、Windows XP，软媒魔方6.08最新正式版下载
                      软媒魔方，全新一代Windows系统增强辅助工具，智能+专业双操控模式，系统故障一键式解决方案，真正实现一键优化、一键清理、一键软件升级。软媒时间，日程提醒、农历天气，so easy！软媒时间，系统美化轻松搞定，桌面好炫酷！简单、好用、好玩，装机必备，从软媒魔方开始。更有网速限速、U盘启动、DNS助手、壁纸美化等功能等你发现。
                        mofang.ruanmei.com

                      • 让你的右键菜单更加智能：FileMenu Tools

                        Windows默认的右键菜单实在是弱爆了，完全是上世纪的产物。还好有了FlieMenu Tools，终于拯救了右键菜单。

                        FileMenu Tools
                          lopesoft.com

                        • 右键菜单清道夫：RightMenuMgr（右键扩展菜单管理器）

                          当你装了一大堆APP后，右键菜单就成了“兵家必争之地”...RightMenuMgr可以帮你给它减减肥，结合FileMenu Tools，一个高效的右键菜单横空出世！

                          RightMenuMgr – 强大的右键菜单管理工具
                          RightMenuMgr 是一款强大的右键菜单管理工具，提供“文件”、“文件夹”、“磁盘”等几乎所有地方的右键菜单管理， RightMenuMgr 附加提供了 IE 右键......
                            appinn.com

                          • 让你时刻充满斗志：ReRe定时休息

                            小众上推荐的一款APP，类似番茄工作法的方式，每隔一段时间强制让你休息。（屏蔽鼠标/键盘操作）

                            ReRe (Relax Reminder)
                            The official website of Relax Reminder
                              rere.appinn.me

                              Windows虽然已经进化到了8.1，但大文件拷贝永远是心中的痛，Fastcopy却能化解这份悲痛！

                              FastCopy
                                ipmsg.org
