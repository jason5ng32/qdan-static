自用 Alfred Workflows 推荐

在 Windows 时代用惯了 Launchy 后，Alfred 就成了自己入 Mac 时的第一款付费应用，而且毫不犹豫的戳了终身版。 经过两年多的大浪淘沙，以及根据需要自己写的几个简单小扩展，如今还保留在我列表中的 Workflows 已所剩不多，分享出来给那些有同样需求的人。

  1. Alfred Maestro

    配合 Keyboard Maestr，可获取 Macros List 名称，并直接调取启用。

    iansinnott/alfred-maestro
    alfred-maestro - An Alfred 2 workflow to execute Keyboard Maestro macros.
      github.com

    • Pinboard Workflows

      无需本地客户端支持，可快速检索线上 Pinboard 的书签列表。

      Pinboard 1.2: Search, Add and Delete Bookmarks (Yosemite support) - Share your Workflows
      Page 1 of 2 - Pinboard 1.2: Search, Add and Delete Bookmarks (Yosemite support) - posted in Share your Workflows: Overview Pinboard workflow is a front end for Pinboard that allows you: search filter by tags add public bookmarks add private bookmarks set a new bookmark as To Read delete a bookmark get the selected text as the bookmark description (see Syntax) interact with: Safari, Google Chrome, Google Chrome Canary, Chromium, Webkit and Firefox First Run When you first run the workflow,...
        alfredforum.com

      • Dash Workflows

        配合 Dash，可以快速离线检索各种代码说明和 API 文档。

        Kapeli/Dash-Alfred-Workflow
        Dash-Alfred-Workflow - Dash Alfred Workflow
          github.com

        • Slackfred

          配合 Slack，可以快速检索用户，文件，跳转频道，设置状态等。

          fspinillo/slackfred
          slackfred - Interact with Slack via Alfred
            github.com

          • Transmit Workflows

            配合 Transmit，读取并连接收藏列表中的服务器。

            ramiroaraujo/alfred-transmit-workflow
            Contribute to alfred-transmit-workflow development by creating an account on GitHub.
              github.com

            • Uninstall with CleanMyMac 2

              配合 CleanMyMac 2，调取其卸载指定的 APP。

              Alfred 2 Workflow List | Search, Install and Share
              Alfred 2 Workflow List, also Alfred 2 Themes and Alfred Tips
                tldrify.com

              • Open in CF

                配合 Safari / Chrome / Firefox，将 Safari 的当前标签页在 Chrome 或 Firefox 中打开。P.S. 我自己用的 Firefox 是 FirefoxDeveloperEditi，如果使用的标准版的 Firefox，请自行修改代码中的进程名。

                24says/Alfred
                Alfred Workflow
                  github.com

                • Units Conversion

                  对各种度量单位进行在线转换，我常用来转换货币汇率。

                  jason0x43/jc-units
                  jc-units - Alfred 2 Workflow for unit conversion
                    github.com

                  • Audio Switch

                    快速在 Mac 音频设备间进行输入输出的切换。

                    sampayo/Alfred-WorkFlows
                    Alfred-WorkFlows - My Alfred v2 workflows
                      github.com

                    • Menu Bar Search

                      通过关键字快速调取和执行当前应用的菜单栏命令。

                      Alfred 2 Workflows | Ted Wise
                        tedwise.com

                      • Top Processes

                        强制关闭指定的应用程序进程，同时还可以查看已运行应用的功耗，进程 ID 等信息。

                        zhaocai/alfred2-top-workflow
                        alfred2-top-workflow - Alfred 2 Workflow: 1) Top Processes Based Memory, CPU, and IO Usage; 2) Get a Glance of System Status
                          github.com

                        • Shadow Killer

                          利用终端命令写的一个开启和关闭系统窗口阴影效果的开关，用于截取无阴影效果的窗口截图。

                        • System File Toggle

                          利用终端命令写的一个简单的显示和隐藏系统文件的开关。

                        • Yosemite Phone

                          利用 Yosemite 新增的基于 iPhone 的通讯功能，可通过 Mac 拨打指定联系人或号码。P.S. 因纯个人使用，所以联系人匹配规则未必适合所有人，可下载后自行修改。

                          24says/Alfred
                          Alfred Workflow
                            github.com

                          • App Store Search

                            快速在 App Store 和 Mac App Store 中检索指定的应用。

                            yisiqi/alfred2-workflow-appstore
                            alfred2-workflow-appstore - This Alfred Workflow is used for searching apps at iTunes or Mac App Store.
                              github.com

                            • Baidu Search

                              顾名思义，虽然 Google 是主力，但偶尔还是会用到这货。

                              Alfred 2 Workflow List | Search, Install and Share
                              Alfred 2 Workflow List, also Alfred 2 Themes and Alfred Tips
                                tldrify.com

                              • Douban Search

                                调取豆瓣快速检索图书、音乐和电影信息。

                                豆瓣 Workflow for Alfred v2

                                  lucifr.com

                                • Taobao Search

                                  调取淘宝快速检索制定商品。

                                  kennyz/alfredworkflow
                                  alfredworkflow - taobao search helper for alfred2 workflow
                                    github.com

                                  • Zhihu Search

                                    调取知乎快速检索话题，问题和人。

                                  • Youdao Search

                                    调取有道词典对中英文进行在线翻译。

                                    aetherwu/youdaoalfred
                                    Contribute to youdaoalfred development by creating an account on GitHub.
                                      github.com
