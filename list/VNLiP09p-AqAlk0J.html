OS XApp应用推荐

码农Mac装机必备

码农们，拿到Mac后要装些什么东西呢？

  1. iTerm

    iTerm2 is a replacement for Terminal and the successor to iTerm.

     

    比系统自带的命令行好用

    iTerm2 - Mac OS Terminal Replacement
    iTerm2 is a replacement for Terminal and the successor to iTerm
      iterm2.com

    • oh-my-zsh

      oh-my-zsh is an open source, community-driven framework for managing your Zsh configuration.

       

      使用Shell的一定要安装Zsh，oh-my-zsh可以帮你非常简单的管理Zsh的配置

      robbyrussell/oh-my-zsh
      oh-my-zsh - A community-driven framework for managing your zsh configuration. Includes 180+ optional plugins (rails, git, OSX, hub, capistrano, brew, ant, php, python, etc), over 120 themes to spice up your morning, and an auto-update tool so that makes it easy to keep up with the latest updates from the community.
        github.com

      • Homebrew

        The missing package manager for OS X.

         

        类似Ubuntu下的apt-get install ，不用再继续介绍了吧。例如：brew install wget

        Homebrew — The missing package manager for OS X
          brew.sh

        • ClipMenu

          Clipboard Manager for Mac OS X.

           

          这个绝对是神器，复制过的东西都会像历史网页一样保存起来，一会你没准就用上了

          ClipMenu.com: A clipboard manager for Mac OS X - ClipMenu.com
            clipmenu.com

          • Chrome

            这个不用描述了吧，现在Chrome基本一统天下了，推荐几个好用的插件吧：Postman，AdBlock，Clearly，JSONView，Pocket

            Chrome Browser
            Google Chrome is a fast, simple, and secure web browser, built for the modern web.
              google.com

            • 红杏

              科学上网神器，你懂的

              User-Agent Switcher
              User-Agent Switcher is a quick and easy way to switch between user-agents.
                honx.in

              • Postman

                非常好用的接口调试工具，一直在用

                Postman - REST Client (Packaged App)
                Supercharge your API workflow with Postman! Build, test, and document your APIs faster. More than a million developers already do.…
                  chrome.google.com

                • AdBlock

                  不怕啦广告

                  AdBlock
                  AdBlock. The #1 ad blocker with over 200 million downloads. Blocks YouTube, Facebook and ALL ads by default (unlike Adblock Plus).
                    chrome.google.com

                  • JSONView

                    如果地址直接返回JSON字符串，就会格式化为非常友好的展现

                    JSONView
                    Validate and view JSON documents
                      chrome.google.com

                    • Sequel Pro

                      MySQL database management for Mac OS X.

                       

                      非常棒的MySQL客户端，智能提醒，导出查询数据什么的变的非常简单

                      Sequel Pro
                        sequelpro.com

                      • JetBrains

                        World's Leading Vendor of Professional Development Tools.

                         

                        写Java？用IntelliJ IDEA吧，写Python去用Pycharm吧，最专业的IDE总有一款适合你

                        JetBrains :: World's Leading Vendor of Professional Development Tools
                        JetBrains, creator of the best Java IDE - IntelliJ IDEA - is a technology-leading software development firm specializing in the creation of intelligent development tools.
                          jetbrains.com

                        • Dash

                          文档中心，想要啥文档离线到本地吧

                          Dash for OS X - API Documentation Browser, Snippet Manager - Kapeli
                          Dash is an API Documentation Browser and Code Snippet Manager. Dash searches offline documentation of 150+ APIs and stores snippets of code. You can also generate your own documentation sets.
                            kapeli.com

                          • Evernote

                            这个也没啥说的，最好的云笔记

                            The workspace for your life’s work | Evernote
                            Bring your life's work together in one digital workspace. Evernote is the place to collect inspirational ideas, write meaningful words, and move your important projects forward.
                              evernote.com

                            • Sublime Text

                              Sublime Text is a sophisticated text editor for code, markup and prose.

                               

                              不需要IDE的时候当然要用Sublime啦，高亮，智能提示啥都有

                              Sublime Text: The text editor you'll fall in love with
                              Sublime Text is a sophisticated text editor for code, markup and prose. You'll love the slick user interface, extraordinary features and amazing performance.
                                sublimetext.com
