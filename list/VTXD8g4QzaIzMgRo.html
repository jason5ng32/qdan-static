《冰与火之歌》里的那些城堡

RT

  1. 临冬城 / Winterfell

    故事开始于史塔克家族的临冬城，在书中这是一座古老、辽阔而高大的城堡，有两道城墙，该图集中表现了其正门和门外的避冬镇

  2. 赫伦堡 / Harrenhal

    曾是河间地王国（铁民统治）的首都，刚建成就被龙焰摧毁。它被描绘成一座阴暗、巨大、受诅咒、鬼魂出没的城堡，带着远古巨龙攻击所残留的痕迹。和临冬城一样（虽然更加巨大），它占地千亩，让画中的船舶显得渺小

  3. 龙石岛 / Dragonstone

    这座城堡是最初的瓦雷利亚征服者用“魔法”建成的，然而这个观念却使得真实的描绘这座城堡成了巨大的挑战。作者在书中写道，这座城堡是由冷却变硬的火山岩组成的，这些岩石呈巨龙的形状，从塔楼到中央巨大的鼓楼莫不如此

  4. 派克岛 / Pyke

    葛雷乔伊的老家，铁群岛首府，书中将其描绘成一系列建在海蚀柱上的高塔与城堡，我并没有致力于表现所有这些城堡，只是表现了海石椅以及与之相连的摇摇晃晃的索桥，其下方则是波涛汹涌的大海。派克岛是海上王冠，一个崇拜淹神的类似维京人的国度

  5. 凯岩城 / CasterlyRock

    兰尼从凯斯德利家族骗来的城堡，英文直译就叫做凯斯德利的岩石，符合西境的地理特征。参照第一种命名风格，兰尼斯特的家堡，不是由石头筑起的城堡，而是“城堡一样的岩石”，很有趣的命名。所以当瑟曦问詹姆“你要石头还是要我”，当提利昂对珊莎说“兰尼斯特才喜欢石头”，我们若不明白凯岩城的真意，就不能准确把握他们的一语双关

  6. 鹰巢城 / The Eyrie

    这是最富奇幻色彩的城堡之一，据说从未失陷过，描绘这座建在巨人之枪半山腰上的巍峨城堡是莫大的快乐，注意那些天牢，它们把囚犯完全暴露在自然的力量之下，唯一的解脱就是纵身一跃投入死亡的怀抱，该图尽量表现了这座城堡的高耸险峻

  7. 君临的红堡 / The Red Keep at King’s Landing

    红堡位于君临城之中，是全维斯特洛最为宏伟的皇家堡垒。透过蒙蒙烟雨，我们可以看到圣贝勒大教堂，这是维斯特洛大陆的西斯敏斯特教堂

  8. 旧镇高塔 / The Hightower at Oldtown

    乔治马丁对这座位于南方的古老建筑惜字如金，这座高塔有七层，每层都是七边形（猜测），建在最下层更为古老的碉堡之上

  9. 黑堡和绝境长城 / Castle Black and The Wall

    这座长城有700尺高，由坚冰筑成，就像英国的哈德良长城或是中国的长城，用以抵御来自北方广袤的荒野之地的入侵。黑堡是长城脚下的一座重要城堡，里面住着誓言终身守卫长城的士兵

  10. 影子塔 / Shadow tower

    影子塔位于长城一端的大山旁，影子塔位于哨兵楼西侧，西桥望东侧。影子塔由丹尼斯·梅利斯特爵士担任指挥官，是长城上三座还有守夜人驻守的城堡之一，剧中没有出现

  11. 东海望 / East Watch By The Sea

    东海望是长城最东边的城堡，位于海豹湾边上狂风肆虐的海滩。由卡特·派克率领的不到二百人驻守在那里，守夜人军团的船只舰队都停泊在东海望，其中既有体积较大的能穿越狭海的船只，也有船身细长的战舰。舰队主要用来在海豹湾巡逻以抓捕那些向野人贩卖武器的走私者。在东海望附近还居住着一些渔民，西边是绿卫堡

  12. 栾河城 / The Twins

    辽阔安静的富饶之地，亦是两代人的墓地与一位母亲的失魂之地。相信剧迷们对于《血色婚礼》所带来的冲击仍未平息，亡灵不死，其复仇之心也将永不泯灭

  13. 奔流城 / RiverRun

    用汉语翻译冰与火之歌的城堡名时，很多只能翻出字面意思，其中内涵没一个能搞得准。 RIVERRUN，河在跑，直接翻译奔流城似乎不错，但 RUN 在英语里面还有运作、掌管的意思，兰尼斯特住在像城堡的石头里，而徒利的家堡也非人工建造，是由三叉戟河给 Run 起来的，（据说是由三叉戟河冲击下堆积的石块所筑成）注意它的位置正好处在河的交汇处，当然猫姨的家族文化也不用多说

  14. 阳戟城 / SunSpear

    恩王国是七大王国中最后对铁王座效忠的国度，血脉、习俗、地理和历史使得多恩人在维斯特洛人中特质明显。“五王之战”初期，多恩没有参加任何一边，但在崔斯丹王子和弥赛菈•拜拉席恩公主订婚之后，阳戟城宣布支持乔佛里国王的事业，马泰尔家族的旗帜是一轮红日为一柄金枪所贯穿，他们的族语是“不屈不挠”

  15. 弥林 / Meereen

    是奴隶湾的一处城邦，坐落于海湾的最北边，斯卡扎丹河畔。它因为奴隶贸易而文明，而第四季也将有重要事件发生于此地，图为弥林的大金字塔

  16. 布拉佛斯 / Braavos

    布拉佛斯是自由贸易城邦中最特殊也最强大的城邦，由厄斯索斯西北端的一系列岛屿组成，正处于狭海与颤抖海交会的地方。从布拉佛斯向南延伸的海岸线被称作布拉佛斯海岸，以宣示布拉佛斯拥有主权。布拉佛斯的统治者被称为海王，因为这些城市的权力和财富全部来自于航海，图为布拉佛斯的泰坦巨人

  17. 高庭 / HighGarden

    河湾地首府，看到这个我就想起巴比伦的空中花园，提利尔家的城堡被称作‘花园’，这正好符合南部肥沃土地，宜人气候下的种植园文化

  18. 风息堡 / Storm'sEnd

    风息堡是七大王国中最坚固的城堡之一，拜拉席恩家族的家座，而在其之前的几千年，是古老的风暴王的王座。城堡据说被编制在城墙内的咒语所保护，以防止魔法透过城墙。第二部凯特的章节。在风暴之威肆虐的边疆地，敢起 Storm's end 这么拽的名字，也只有敢和神对抗的人做得出来
