十大 Android 平台开源自由软件

Android 自诞生之日起就与开源有着密不可分的关系， Google 通过 AOSP ( Android Open Source Project ) 将这一系统变成了用户数最多的 Linux 衍生产品。

 

AOSP 是开源的没错，但是不代表上层应用也是开源的。在移动端，依然有人坚持做开源自由软件。本轻单为你盘点 Android 平台的十大开源自由软件。

  1. Firefox

    简洁，易用，反应灵敏，最重要的是可以安装扩展，简直神器！

     

    另外还可以搭配桌面端 Firefox 对移动端网页进行 USB 调试。

    Mobile/Platforms/Android - MozillaWiki
      wiki.mozilla.org

    • Orbot

      手机上的 Tor ，你懂的。下面是官网链接，请自备梯子。

      Tor Project: Android Instructions
        torproject.org

      • VLC

        音乐、视频播放器，和桌面端的 VLC 一样强大。

         

        支持 ARMv6, ARMv7, MIPS 和 x86 架构。支持硬解和软解，但是硬解也要看具体设备。支持字幕，支持在线流媒体。当然一些功能也在逐步打磨中。

        VideoLAN - VLC media player for Android™
        VLC media player for Android™
          videolan.org

        • K-9 Mail

          简洁的邮件客户端。如何你在手机上安装了 GPG 应用，它可以为你的邮件签名、加密，非常方便。具体教程请移步下面的轻单：

          手机上的GPG加密其实更易用 | 轻单
          本文介绍在Android平台上使用 GPG 对邮件、文件进行加密、签名的方法。 GPG 加密，想必你应该听说过吧？安全性当然是毋庸置疑，但是为什么没有普及到广大群众呢？可能你会说麻烦。其实 GPG 比想象中更简单易用。
            qdan.me

          • Barcode Scanner

            扫描或生成二维码的小工具。

            zxing/zxing
            zxing - Official ZXing ("Zebra Crossing") project home
              github.com

            • DAVdroid

              Android 4.0+ 平台。用来同步你的联系人和日历，你可以通过它和你自己的服务器进行同步，保障数据安全。

              DAVdroid: What is DAVdroid?
                davdroid.bitfire.at

              • PPSSPP

                PSP 模拟器。需要你自行导入游戏的 ISO 镜像。

                hrydgard/ppsspp
                A PSP emulator for Android, Windows, Mac, Linux and Blackberry 10, written in C++. Want to contribute? Join us in #ppsspp on freenode (IRC) or just send pull requests / issues.
                  github.com

                • ownCloud

                  你的私有云存储，可以将文件放在你的 VPS 上，用它来同步。

                  owncloud/android
                  android - The ownCloud Android App
                    github.com

                  • OsmAnd~

                    离线地图和导航。可以在设置里启用插件，增强功能。

                    osmandapp/Osmand
                    Osmand - osmand
                      github.com

                    • AFWall+

                      Android 上的 iptables 前端，接棒已停止开发的 DroidWall 。

                       

                      支持导入/导出规则，可设置多帐户，系统应用高亮显示等。

                      ukanth/afwall
                      afwall - AFWall+ (Android Firewall +) - Firewall for Android
                        github.com
