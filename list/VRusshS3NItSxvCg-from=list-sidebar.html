2014年被严重低估的11部电影

  1. 性本恶

    这片充满负面的争议和压倒性的差评其实也在情理之中，神秘高逼格大师品钦的文学作品本就是根难啃的硬骨头。

    性本恶
    （2014）
    6.7
    早在2010年的12月，安德森就有意改编托马斯·品钦的《性本恶》，不过由于另一部新片《大师》的资金到位较早，《性本恶》暂时被搁置了起来。《性本恶》有一个类似钱德勒侦探小说的外壳，洛杉矶私家侦探多克与前女友莎斯塔不期而遇，并从她那听说了一个无从佐证的故事，一桩离奇绑架案，受害者——她现在的情人——是一个“腰缠亿贯”的房地产商。多克明知道“爱情”是个含混不清、随时会惹出一串麻烦的词儿，还是鬼使神差地撞进了莎斯塔的故事，而且愈卷愈深，冲浪手、皮条客、贩毒者、摇滚乐手、犯了命案的高利贷者，各色人等，若隐若现……

  2. 锡尔斯玛利亚

    《锡尔斯玛利亚》和《鸟人》的主题有点异曲同工之妙，前者以绝妙的台词混淆，后者以别具一格的运镜手法，都以电影镶嵌戏剧的形式探讨了演员的职业状态和日常生活。

    锡尔斯玛利亚
    （2014）
    7.6
    上了年纪的演员玛利亚（朱丽叶·比诺什 饰），年轻时曾饰演“西格德”一角，西格德是一个充满野心，具有致命魅力的女孩，吸引了一个名为海琳娜的成熟女人并导致她最终自杀。这个角色让年轻的玛利亚获得了成功，跻身名流。多年之后，年过40的玛利亚对自己的年龄格外敏感。此时，玛利亚将再次出演成名作，不过角色由迷人的西格德变为自杀而死的海琳娜。而好莱坞问题少女艾利斯（科洛·莫瑞兹 饰）将接手西格德的角色。玛利亚对艾利斯由开始的不屑一顾逐渐转为依赖，如同海琳娜对西格德的迷恋，然而这份危险的痴迷却伤害了玛利亚与她唯一的朋友，私人助理薇伦汀（克里斯汀·斯图尔特 饰）的亲密关系。

  3. 狐狸猎手

    不过分显山露水异常冷峻隐忍的叙事手法，看似不过是还原了个真实的摔跤故事，实际暗涌暧昧可怖的气息和强大的精神诉求，以及夹裹着厚重而深刻的人性意图，被无限扩张扭曲的欲望构建出了一个披着金色外衣实际不堪一击的“黑色美梦”。

  4. 冬眠

    锡兰精心刻画了一个包含了社会、家庭和个人问题的影像世界，你会觉得异常的美妙，看似裹着一件单薄乏味的外衣，实际有着坚实厚重的精彩内核。

    冬眠
    （2014）
    8.1

    • 主演: 哈鲁克·比尔根纳尔 / 梅丽莎·索岑 / 迪米·阿克贝 / 艾贝尔克·佩克詹

  5. 修女伊达

    有时候剧情并不能作为衡量一部电影优劣程度的唯一标准，甚至有心淡化的意图能获得某种更微妙的感官体验，不需要作过多的阐释，就像一幅牛逼的简笔画，刻意的着色反倒是画蛇添足失去了意境，看似寥寥几笔实际有着更加值得玩味的思想和主旨。

    修女伊达
    （2013）
    7.7
    1962年的波兰。安娜是一名在修道院长大的孤儿。在宣誓正式成为修女之前，她离开了修道院，去见世上剩下的唯一亲人旺达。旺达告诉安娜，她是犹太人。两名女子踏上了一段旅程，不仅将发掘属于她们家族的悲剧，而且将找到她们自己真正的归属。她们开始质疑之前的信仰，质疑之前生活中所信任的一切。安娜和旺达都想继续活下去，但是当中只有一个人能做到。

    • 主演: 阿伽塔·库莱沙 / 阿伽塔·特泽布霍夫斯卡 / 乔安娜·库里格 / Dawid Ogrodnik

    豆瓣电影

  6. 游客

    相比《消失的恋人》，《游客》更加贴近现实，生活即是如此的即视感，人性的真相，婚姻的危机，关系的处理，在看似非常日常简易的镜头语言里进行了有力深刻的探讨，抛出了道德的困惑，就像看似很稀松平常笼罩在头顶的一片乌云，却有着让人深陷愁思的强大力量。

    游客
    （2014）
    7.5
    阿尔卑斯山滑雪佳地，一瑞典家庭在就餐时遇到了雪崩，丈夫托马斯只顾自己逃命不救妻孥，引发了家庭危机…… 一个瑞典家庭来到了法国的阿尔卑斯山区度假。当天风和日丽，景色无比养眼，然而当他们在山脚下的一个餐馆用餐时，一场突如其来的雪崩打破了这一切。游客们四处逃窜，母亲艾芭叫喊着她的丈夫托马斯的名字，试图保护他们的孩子。然而没想到的是，托马斯却在慌乱之下抛妻弃子，自顾着逃命去了……尽管最终躲过了雪崩，但这一家人的关系却出现了巨大的裂痕。托马斯究竟怎么样才能挽回妻子的心，并且在孩子们的心目中重塑他的威信？

    • 主演: 约翰内斯·昆科 / 丽莎·洛文·孔斯利 / 克里斯托弗·海维尤 / Clara Wettergren
    • 类型: 剧情
    豆瓣电影

  7. 奇迹

    在以商业片和烂片为主打的当下，能看到这么一部质朴的电影实在太难得了，严父慈母带着孩子们以养蜂为生的清贫生活，虽在现代文化和外来因素的影响下，引起了波澜，辞旧迎新融入新时代或将成为必然结果，但家人间的温情、自然给予他们的技能和价值观将会永远留存。

    奇迹
    （2014）
    7
    意大利的年轻美女导演爱丽丝·洛尔瓦彻虽然刚刚三十出头，第二部作品《奇迹》就已成为意大利唯一代表，出征今年的金棕榈之争。 影片讲述了一对夫妻带着四个女儿，在靠近海边的独立小农庄里，依靠养蜂过着自给自足的独立生活，远离城市和社会人群。然而有一天，一个德国少年犯在重新融入社会计划下进入到她们的生活里，同时，一个名叫《神奇村庄》的真人秀节目也在这里选景拍摄，片中有美丽性感的主持人……这一切新鲜事物，打乱了她们艰苦却简单平静的生活，尤其是进入青春期的大女儿吉索米拉，对外面的缤纷世界充满渴望。 这是一部奇特的实验电影，你很难将它具体归类，神秘主义？带有神秘色彩的现实主义？再或者……？似乎每一种说法都有点关联，又不尽相同。如果说大多数影片看完后，观众会有强烈的表达愿望，但它却是让你恍惚、困惑、无从说起的影片。这也注定了它将在媒体影评人中引来诸多分歧。

    • 主演: 莫妮卡·贝鲁奇 / 阿尔芭·洛尔瓦彻 / 玛格丽特·提塞尔 / 安德列·汉尼克

  8. 宿敌

    整体干脆利落点到即止，毫不拖泥带水值得细品，线索细节耐人推敲，老妈的话一语破的揭开迷雾，欲望引发的疯狂臆想，最大的敌人即是自己。

  9. 沼泽地

    一流的“侦探片”，构图、运镜、摄影、配乐、细节都非常的讲究和细致，悬疑紧张的氛围引人入胜，案件构思缜密，抽丝剥茧层层递进拨开迷雾，放入特殊的时代背景，引入对于社会和人性的思考。

    沼泽地
    （2014）
    7.5
    在20世纪80年代初的西班牙，为了调查少女失踪案，两位思想观念完全对立的警察从马德里被派往南边的一座偏远小镇，这座小镇位于瓜达尔基维尔流域的沼泽地上。在这样一个依然保留着根深蒂固的陈旧思想的社会中，他们将面临一个残忍的杀人凶手。

  10. 再见语言

    这部的逼格可能要远远甩开其它片子，心照不宣戈达尔已经上升到了另一境界，虽然看完可能会摸不着头脑云里雾里，但一点也不妨碍对其敬意的滋生，作为新浪潮时期的重要旗帜，现今仅剩的几根独苗之一，还能保持强劲的创作力，有着另辟蹊径的冒险精神，永远走在先锋而充满哲思的探索道路上，是值得让人永远保持钦佩和敬仰的大师。

  11. 苦路十四站

    这片的一大特点在于导演透过简洁有力的镜头语言和别具一格的宗教象征意义上的叙事形式，全片被分为十四个小章节，每个章节基本以固定镜头的对话形式呈现，危机、隔阂、分歧、荒诞在看似寻常的日常中很有力道的流淌于光影中，非常的冷峻而深刻，并以耶稣的名义隐喻形象的反映出现实中玛利亚的生活，体现了信仰的多样性之间产生的矛盾和对立，对信仰的魔性进行了尖锐深刻的呈现和讽刺。

    苦路十四站
    （2014）
    7.8
    自從開始上天主教慕道課以來，恪守教義的少女瑪麗亞便遠離了流行文化，抵抗撒旦化身的種種誘惑。長大之後，嚴厲的母親仍以絕對威權的方式，約束她的生活起居與交友狀況。虔誠的瑪麗亞不但努力達到要求，為了讓罹患自閉症的弟弟能開口說話，她暗自做了一個危險的決定。 《少女的歧禱》以工整完美的形式，呼應片中角色的極端行徑。全片十四則標題，借典自耶穌赴難的章回短篇；十四段長鏡頭，築架出儀式般的嚴謹結構，再再讓人聯想起一九五○年羅塞里尼執導的《聖法蘭西斯之花》。不同的是，本片看似冷靜側寫瑪麗亞追求信仰而犧牲奉獻的心路歷程，實則以強烈批判性的諷喻口吻，探討極端的信仰狂熱如何成為威權高壓統治的變種，魄力驚人。
    • 导演: 迪特里希·布鲁格曼
    • 主演: Lucie Aron / 安娜·布鲁格曼 / Michael Kamp / Moritz Knapp
    • 类型: 剧情
    豆瓣电影

  12. 【彩蛋】被严重高估的电影：星际穿越

    在不少人心目中，诺兰已经妥妥滴步入了神坛，但还是想冒着往死喷的危险说说我的个人感受，用脚趾头想也知道应该会被淹死在诺兰铁粉们的唾沫星子里。这片是在朋友的极力推荐下满心期待的跑去影院消化了（其实和窝在家里弄个投影仪看的效果完全没差），结果隐忍硬撑了将近漫长的三小时，腰酸背痛一点也不夸张，字幕出现时总算松了口气，磨磨唧唧+狂扔理论+煽情戏码，怀揣着恒久不变的商业片主旋律核心价值观，真是麻木到累觉不爱了，现在再在脑海中回想真没啥深刻的印象，只能套句网友笼统的话来为这片做个总结：科幻版的爸爸去哪了！

    星际穿越
