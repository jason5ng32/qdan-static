从零开始，Sketch学习五部曲

仅作自学参考~你的一万小时开始啦

  1. 基本了解 Sketch 是什么 -- 0.25小时 / 一万小时

    这是 Sketch 的开发者 Pieter Omvlee 发布的几个简单教学视频，看完它，建立基本的了解。

    Pieter Omvlee
      youtube.com

    • 看完一遍《Sketch 3 中文用户手册》 -- 20小时 / 一万小时

      一字不拉的看完 Sketch 3 中文用户手册 ，对照着将里面提到的所有功能摆弄一番。

      不要偷懒了，这是最简单粗暴的学习方法，而且最多花掉20个小时的时间，我计算过。

      在 Sketch 里所有概念都很容易理解，图层啊蒙版啊，你会发现原来是这么简单的一回事，

      里面各种工具都并不重型，但绝对满足需求，而且 Sketch 所有操作方式都太人性了，异常顺手。

       

      附：

      官方原版说明书《Sketch 3 Documentation》http://bohemiancoding.com/sketch/support/documentation/

    • 研究&临摹别人的作品 -- 20小时 / 一万小时

      去研究别人的 Sketch 文件，下载一下免费的 Sketch 资源，看看人家的文件，是怎么管理图层，怎么使用一些工具，怎么实现一些效果，你会受益良多。

       

      例如：

      http://www.sketchappsources.com/all-free-sources.html 超过400个免费的 Sketch资源下载，模板、设备、图标，非常全面

      http://sketchmine.co/ 一位法国设计师收集的Sketch 资源，分类清晰，他同时提供收费的移动App设计课程

      http://www.teehanlax.com/tools/iphone-sketch-app/ 大名鼎鼎的 Teenhan+lax 创作的iOS 7 GUI 套件

      http://www.sketchgems.com/ 一个用 Sketch 创作UI 套件和图标的团队网站，部分免费

      http://brilliantsketch.com/ Dribbble 上一位印尼设计师的作品，有资源，有技巧

    • 开始自己做点东西 -- 1000小时 / 一万小时

      好了，开始自己做点东西吧。

      如果完全没有头绪，我建议你先从小小的东西开始，比如画一个图标，画一个简单的app页面，你可以自己创作，也可以模仿出色的作品，研究每个效果是如何实现的。

       

      http://www.aegeank.com/sketchactive/

      这里360个图标，也就是我做完上面说的这三件事开始画的，仔细看，其实 iOS 7 的没有技术含量，因为完全是照着画的，而第二套 Businese 的其实挺丑的，图标大小不统一，风格不统一，处理的很粗糙，我现在都不好意思拿出来 > < 但是从 Sweet，coffee & tea 那套开始我找到感觉了，之前用户手册里还有些不太理解的东西当时也都完全明了了，开始把自己的想法倾注在作品中。再后面的几套就已经可以玩一样的很熟练很高效的使用 Sketch了。

    • 温故而知新，让设计更有内涵 -- 8959.75小时 / 一万小时

      有了一些实战经验，我们再来补充一些理论知识，有一些很棒的设计师博客，他们分享了各种技巧，帮助你去master Sketch。

       

      http://hicc.me/design-with-sketch/ Sketch 3 的新功能

      http://sketchtips.tumblr.com/ Sketch 的开发团队 Bohemian Coding 在tumblr 上维护的技巧博客（注：博客已在一年前开始停更）

      https://medium.com/sketch-app Medium 上关于 Sketch 的collection

      https://designcode.io/ 一位很棒的 UI 设计师 Meng To 创作的学习用Sketch做设计+用Xcode编程的书

      http://www.sketchcasts.net/ 一个葡萄牙设计团队创作的 Sketch 教学视频，每周更新，需付费观看

      https://github.com/sketchplugins/plugin-directory Github上由 Bomber Studios制作的一系列插件，帮助你更高效的使用Sketch。

      http://www.sketchcn.com/sketch-plugins.html

       

      Github 上由 shahruz 制作的一个叫 Sketch Toolbox 的 Mac 应用，集合了目前绝大多数的 Sketch 插件，在里面可以一键安装和使用。
