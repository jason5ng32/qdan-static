快速了解中国最后一位水墨动画大师的作品

前几天，被誉为“中国最后一个水墨动画大师”的动画艺术家马克宣因病在上海逝世，他曾参与制作《大闹天宫》、《小蝌蚪找妈妈》、《哪吒闹海》、《三个和尚》等众多经典国产动画片，每一部都是我们童年的经典，这个轻单整理了他曾参与创作的作品。

  1. 担任动画师作品

  2. 《大闹天宫》

    中国动画界开天辟地的作品，是中国动画史上的丰碑，也被誉为中国动画难以逾越的一部作品，作为中国动画片的经典影响了几代人

    大闹天宫
    （1961）
    9.2
    话说在东土傲来国有一座花果山，山上有一尊石猴吸收日精月华化身为一只神猴（邱岳峰 音），统领着山中的猴子猴孙。为求得一件称心的宝贝，神猴孙大圣潜入龙宫，强硬求来大禹治水时的定海神针如意金箍棒。东海龙王（毕克 音）心有不甘，于是上天将此事诉诸玉帝（富润生 音）。玉皇大帝命令太白金星（尚华 音）下界招安，许以爵位。不知有诈的孙大圣欣然前往，却发现只是负责养马的弼马温。得知受骗的猴王反下天庭，与天兵天将在花果山展开大战…… 本片根据古典名著《西游记》中的故事改编，前后历时4年创作，并荣获1962年捷克斯洛伐克卡罗维发利国际电影节短片特别奖、1978年英国伦敦国际电影节年度杰出电影奖、1982年厄瓜多尔第四届国际儿童电影节三等奖、第二届中国电影“百花奖”最佳美术片奖。©豆瓣
    • 导演: 万籁鸣 / 唐澄
    • 主演: 邱岳峰 / 富润生 / 毕克 / 尚华
    • 类型: 动画 / 奇幻
    豆瓣电影

  3. 《小蝌蚪找妈妈》

    中国第一部水墨动画片，，打破了动画片单线平涂的传统形式，将水墨画与动画技术结合，创造出令人叹为观止的艺术魅力，在国际上多次拿奖

    小蝌蚪找妈妈
    （1961）
    8.6
    本美术片是中国第一部水墨动画片，集体编导，艺术指导为特伟，美术指导为钱家骏。影片拍竣后的十多年，在多个国际电影节上获得荣誉，并在1962年第一届中国电影百花奖上荣获最佳美术片奖。 青蛙妈妈产下的宝宝们慢慢长出尾巴变成一群小蝌蚪，它们在水塘里游啊游，遇到两只小鸡，虽然彼此不认识，可是也很快成了好朋友。小鸡和妈妈的亲密让小蝌蚪们羡慕不已，它们决定去找自己的妈妈，路上，它们从虾爷爷那里知道了妈妈的模样，以为能顺顺利利地把妈妈找到。可是找起来，还真不是一件简单事。©豆瓣
    • 导演: 特伟 / 钱家骏 / 唐澄
    • 主演:
    • 类型: 动画 / 短片
    豆瓣电影

  4. 《牧笛》

    世界第二部水墨动画片。该片以牧童寻牛的故事为明线，以笛声为暗线，将一幅幅中国水墨以动画的形式展现在观众面前，展现了美妙的中国风光

    牧笛
    （1963）
    8.8
    初夏时分，蝉鸣成片。蜿蜒的溪水旁，柳条垂丝，随风摇动。在小路的尽头，一头水牛信步来至画的中央。牛背上，穿着短裤短褂的牧童悠然自得吹响竹笛，笛声飘至郊外每一个角落，好一幅天然闲适的画卷。水牛没入水中，享受难得的清凉。牧童斜倚树丫，一条腿自在地晃荡着，他先与鸟儿赛着旋律，之后便沉沉进入梦乡。梦中，他恍惚听到牛儿的呼唤，那顽皮的生灵跳跃玩耍，快乐非常，它自顾自朝着山的尽头踱去，全然不理牧童的呼唤。放眼望去，层峦叠嶂，雾霭重重，少年的牛儿所在何方…… 本片荣获1979年丹麦第三届欧登塞国际童话电影节金奖。©豆瓣
    • 导演: 特伟 / 钱家骏
    • 主演:
    • 类型: 动画 / 奇幻 / 短片
    豆瓣电影

  5. 担任动画设计作品

  6. 《三个和尚》

    影片根据中国民间谚语改编，与现实生活联系紧密，几代人珍贵童年回忆

    三个和尚
    （1981）
    8.6
    三个和尚没水喝的民间故事家喻户晓，本动画片在故事上有一定的新编，形式也异常鲜活好玩。 红衣小和尚不远千里来到一座高高耸立在山顶上的寺庙，立刻做起快乐勤劳的小主持，虽然挑水只能到山下很是辛苦，他仍做得乐此不疲。不久，来了一个蓝衣瘦高和尚，下山挑水的重任于是落在他的身上，很快，瘦高和尚因为该问题同小和尚发生冲突，两人开始抬水喝。后来，他们又迎来一个黄衣胖和尚，胖和尚刚入寺庙，就把水缸里的水喝了个光，小和尚和瘦高和尚一下不乐意了，三个和尚为如何抬水吃水问题大打出手。老鼠咬断香烛引发的一场大火，让三个和尚明白了一个道理。©豆瓣
    • 导演: 徐景达 / 马克宣
    • 主演:
    • 类型: 动画 / 短片
    豆瓣电影

  7. 《哪吒闹海》

    我国第一部大型彩色宽银幕动画长片，在 1980 年成为第一部在戛纳参展的华语动画电影

    哪吒闹海
    （1979）
    8.7
    商末时期，陈塘关总兵李靖的夫人怀胎3年6个月，最终生下一个肉球，肉球中跳出一个伶俐可爱的男孩。李靖原当他是妖孽欲杀之而后快，幸太乙真人前来收之为徒，赐名哪吒，并赐以乾坤圈和混天绫两件宝贝。某年，天下大旱，甘露未降。哪吒和家将到东海洗澡玩耍。他用混天绫搅动海水，撼动龙宫，惹出巡海夜叉李艮和龙王三太子敖丙。几番言语不和，哪吒杀此二人，由此闯下通天大祸。四海龙王齐聚陈塘关索命，为救一方百姓，哪吒决定交出生命…… 本片根据《封神演义》故事改编，并荣获1980年电影百花奖最佳美术片奖；文化部1979年优秀影片奖、青年优秀创作奖；1983年菲律宾马尼拉国际电影节特别奖；1988年法国布尔波拉斯文化俱乐部青年国际动画电影节评委奖和宽银幕长动画片奖。©豆瓣
    • 导演: 严定宪 / 王树忱 / 徐景达
    • 主演: 梁正晖 / 邱岳峰 / 毕克 / 富润生
    • 类型: 冒险 / 动画 / 奇幻
    豆瓣电影

  8. 《天书奇谭》

    中国动画中的又一部里程碑似的、绝对顶级的、足以彪炳史册的伟大作品，从故事情节的编排到人物造型与性格的设计再到动画技术的运用都堪称无以伦比，它虽然没有《大闹天宫》《哪吒闹海》在国际上那么知名，但很多人还是会在自己心中将其列为中国动画的第一

    天书奇谭
    （1983）
    8.9
    天庭有珍奇书籍“天书”，被锁在石龛门里三千年。看守天书的袁公对天书的内容好奇不已，他乘无人之际，偷偷拿出观看。原来，天书上记录了一百〇八条法术，袁公偷偷下凡，把天书上的内容刻在石壁上。但本意造福人类的袁公此举却触犯了天条，遭到了玉帝的惩罚。 炼丹炉里的仙丹不慎被三条狐狸偷吃了，她们变成精下凡骗人钱财，还假扮仙姑和官府勾结，欺压百姓，祸害人间。袁公便指引天赋异禀的蛋生——一个从蛋里跳出来的孩子修习天书上的法术，和狐狸精斗智斗勇，为民除害。©豆瓣
    • 导演: 王树忱 / 钱运达
    • 主演: 丁建华 / 毕克 / 尚华 / 苏秀
    • 类型: 动画
    豆瓣电影

  10. 《山水情》

    被公认为水墨动画至今无人超越的典范，其诗一样的气质、幽远清淡的画面已达到天人合一的境界

    山水情
    （1988）
    9.2
    本片是由上海美术电影制片厂1988年出品的水墨动画电影。讲述了老琴师在归途中病倒在荒村野渡口，渔家少年留老人在自己的茅舍里歇息，老人感到宽慰。翌日，老人病体康复，取出古琴，弹奏一曲，琴声把少年引到他的身边。少年学艺心切，老人诲人不倦，两人结为 师徒。秋去春来，少年技艺大进，老人十分欣喜。但慰藉之余，思虑如何使弟子更上一层楼。一日，老人偶尔看到雏鹰离开母鹰独自展翅翱翔的情景，豁然开朗。于是携少年驾舟而去，经大川而登高山，壮美的大自然，使少年为之神往。临别时，老琴师将心爱的古琴赠送给他，然后独自走向山巅白云之间。少年遥望消失在茫茫山野中的恩师身影，顿时灵感涌起，盘坐在悬崖峭壁之上，手抚琴弦，弹奏着心中之曲，倾吐着对人生的赞美，悠扬的琴声在山间回响。 本片人物、场景设定邀请了著名国画大师吴山明和卓鹤君先生，这也为本片的美术打下了坚实的基础。此外，《山水情》被公认为水墨动画至今无人超越的典范，其诗一样的气质、幽远清淡的画面已达到天人合一的境界。©豆瓣
    • 导演: 特伟 / 阎善春 / 马克宣
    • 主演:
    • 类型: 短片 / 动画
    豆瓣电影

  11. 《超级肥皂》

    影片利用简单背景、人物线条、情节进行很有深度的表达，不但涉及体制化题材，还有三个和尚客串

    超级肥皂
    （1986）
    8.7
    熙攘马路边的小摊上，小胡子商人正在卖超级肥皂，只见他用肥皂轻轻一擦，一块脏布变成了白布，惊叹不已的行人买回去一试，果然什么颜色都能擦成白的，于是一传十十传百，各种身份各种国籍的人都加入了购买小胡子肥皂的队伍中来，很快，路边摊变成了“超级肥皂店”、“超级肥皂公司”。在这过程中一个带着孙女一直持观望态度的老头，最终也掏出钞票买了一块，他和孙女当然也变成了雪白的人。 当大伙看到喜庆的婚嫁队与哀伤的出丧队完全一样时，才感觉只有白色的世界十分荒唐。这时，一个身穿红裙的小女孩蹦蹦跳跳地出现在大伙面前，好奇的大伙跟着她一路向前，发现她是卖超级肥皂的小胡子商人的女儿，而“超级肥皂公司”，现在变成了“超级颜料公司”。©豆瓣
    • 导演: 徐景达 / 马克宣
    • 主演: 动画设计：庄敏瑾
    • 类型: 喜剧 / 动画 / 短片
    豆瓣电影

  12. 《十二只蚊子和五个人》

    影片构思独特，富于哲理，导演采用抽象化、象征化、符号化的表现方法以及细节简化、背景淡化的拖法来突出人物，增强了艺术效果

    12只蚊子和5个人
    （1992）
    8.3
    盛夏来临，蚊蝇肆虐。在某处，离地面不远处的空中，12只蚊子整齐地排出阵型，等待猎物的出现。不久，一名高个儿男子出现。猝不及防之时，蚊子军团呼啸而下，对男子进行围攻，他从头到脚浑身上下全部中招。男人想了想，觉得一人挨咬实在太亏，他算了算蚊子的数量，瞬间想出一个好办法。男人又拉来一个敦实的男子，这时蚊子军团再次发起进攻，不同的是每人只遭到6只蚊子的攻击。高个男人一阵窃喜，敦实男人则想法减轻损失，于是又找来一个牺牲品。如是在三，直到第五个男人出现，绝对平均的均衡被彻底打破了…… 本片荣获第二届上海电影节科教类奖。©豆瓣

    • 类型: 动画 / 短片

    我的朋友小海豚
    8.2

  16. 《大扫除》

    大扫除
    7.1

  17. 《新装的门铃》

    新装的门铃
    （1986）
    7.9
    男人新装了一个门铃，得意洋洋地欣赏了一番，关起门来等人按响门铃。门外有了脚步声，他的心怦怦直跳，可是无论是楼上的女邻居还是拍皮球玩的小朋友，都只是在他门外稍作停留。终于，来了一个找他的人，可那家伙竟然咚咚敲起门来，男人自然不理会他。 同样不识趣的邮递员来了，用力擂起门来，气得脸发青的男人和他对砸了一会，打开了门。接过邮递员递来的汇款单，男人请邮递员按一下他新装的门铃。本以为这下能听到清脆的门铃声了，不想男人只听到摩托起动的声响。©豆瓣
