一览 Windows 图形界面发展史

关于 Microsoft Windows 就不用多介绍了，相信每个人都用过，这篇轻单是《一览 Mac OS 图形界面的发展史》的姐妹篇，带你一览巨硬公司的 Windows 系统用户界面发展史。

  1. MS-DOS

    1981 年 7 月发布，是微软公司在个人计算机上使用的一个操作系统载体，命令行界面。

  2. Windows 1.0

    1985 年 11 月发布，微软第一次对个人电脑操作系统进行用户图形界面的尝试，基于 MS-DOS 操作系统，可以显示256种颜色，窗口可以任意缩放，当窗口最小化的时候桌面上会有专门的空间放置这些窗口（其实就是现在的任务栏）。

  3. Windows 2.0

    1987 年发布，一套基于 MS-DOS 操作系统、类似 Mac OS 图形用户界面的 Windows 版本。这个用户界面较 Windows 1.0 有着更多的功能，窗口可以重叠，可以改变大小，可以最大化和最小化，基本近似 Windows 3.0 。

  4. Windows 3.0

    1990 年 5 月发行，这一版用户界面做了重大改变：MS-DOS 的文件管理程序被基于图标的程序管理程序以及基于列表的文件管理程序取代，并雇佣了图标女皇 Susan Kare 设计了 Windows 3.0 的图标并统一了图形界面的风格。之后此版本成为微软第一个在世界上获得成功的 Windows 版本。

     

    One more thing, 著名的 Windows 蓝屏也从这个版本出现了。

  5. Windows 95

    1995 年 8 月推出，微软划时代的版本，重新设计了整个用户界面，带领普通人电脑操作进入完全图形界面时代。自此有了经典的开始菜单、右键弹出菜单、“复制”/“粘贴” 、图标状态、关闭窗口等，奠定了微软视窗操作系统的统治地位地址，结束了桌面操作系统间的竞争。

  6. Windows 98

    1998 年 6 月发行，基于 Windows 95 编写，图标风格和 Windows 95 几无二致，不过颜色支持得更多了，支持超过了256色的图标。另外开始支持活动桌面，集成的 IE 浏览器开始垄断之路。

  7. Windows Me

    2000 年 9 月发布的千禧年版，也是 Windows 9x 系列中最后一个产品（介于 Windows 98 和 Windows XP间的过渡产品），开机加载 Windows 图形界面前隐藏了加载 DOS 的过程，并提供了与 Windows 2000 一致的界面外观。其他一些用户界面方面的改进也为之后的 Windows XP 提供了一些借鉴。

  8. Windows XP

    2001 年 10 月上市，微软又一个划时代版本，随后统治了操作系统市场长达 11 年之久，并成为有史以来销量最大及最常用的操作系统。

     

    开始支持上百万颜色，将传统的“8位”色系界面带到了“塑胶”色系的图形界面。另外新的图形用户界面“月神”（Luna）开始采用新的窗口标志、新式开始菜单，并加入了许多视觉效果，另外还支持主题换肤，用户可以自由改变整个界面的外观与风格。

     

    那张蓝天白云默认桌面背景成为世界上辨识度最高的照片，也成为一代人的经典记忆。

  9. Windows Vista

    2007 年 1 月发行，用户界面上新增了透明磨砂玻璃的 Windows Aero 效果、 3D 、动画及实时缩略图预览等，并集成新的包括 Segoe UI 在内的字体，默认支持 ClearType 效果，字体更清晰边缘更平滑。但同时对硬件配置要求也较高。

  10. Windows 7

    2009 年 10 月发行，基于 Vista 改进，除了功能上的改进外在用户界面上也下足了功夫，如重新设计了任务栏，改进了 Aero 效果，如 Aero Glass 更加透明华丽且省资源。

  11. Windows 8

    2012 年 10 月上市，用户界面改动较大的版本。新增平板 Metro UI 界面，传统桌面部分移除了经典的开始菜单，取消了之前 Windows 7 里面的 Aero Glass 等效果，文档资源管理器及办公应用开始使用 Ribbon 效果，Windows 桌面小工具也被移除。

     

    因为遭到众多用户反对，微软在随后的改进版本 Windows 8.1 任务栏上又恢复了开始按钮，可点击开始按钮进入开始屏幕。

  12. Windows 10

    将于 2015 年 6 月发行，基于 Windows 8.1 开发，界面上 Windows 10 采用了全新的开始菜单，并将 Windows 8.x 系统中的开始界面集成至菜单当中。另外 Modern 应用也允许在桌面以窗口化模式运行。
